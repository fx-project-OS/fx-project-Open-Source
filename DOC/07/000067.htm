The password should be changed regularly for security reasons. Passwords should not be names, birthdays, or other known information about a user, but should be random numbers and letters.




Change password and activate user

Here you can change your password.

For security reasons, the old password must be entered in the top line and the new password in the bottom two lines.

NOTE:
By a password change, the user is automatically activated if this was locked or inactive!



Field Descriptions

Old Password:

The "old" password must be entered in this field for security reasons.

New Password:

A personal password must be entered in this field. For your own security, the password should be at least six characters long, should not contain any "$" special characters, etc., and should not consist of any easily guessable sequences of letters or numbers such as "123456", "654321", "qwertz" etc.

The password is stored internally in encrypted form, so that a new password must be requested if it is "forgotten".

Repeat New Password:

Please enter the new password again to avoid typos.


Change

A click on this button changes the password permanently.
