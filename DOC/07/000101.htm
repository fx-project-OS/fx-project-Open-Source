Reporting > Materials > Material Report

Listing of all defined materials including their assignment to tasks and detailed consumption


Listing of all defined materials including their assignment to tasks and detailed consumption. The materials to be listed can be conveniently restricted via a selection mask.




Material selection

Search criteria can be used to limit the number of material lists to be displayed. As search criteria, for example, article numbers, material names, assigned projects, material types, usage types, storage locations and/or start or end date can be specified.



Field Descriptions

Int. Artikelnummer:

If a number is entered here, only materials are listed which contain this number completely or partially.

Materialname:

If a name is entered here, only materials are listed which contain this name completely or partially.

Project:

If a project is selected here on the one hand only immediate materials are indicated, which are assigned directly, or on the other hand only consumption or time materials, which are assigned indirectly by an already occurred consumption posting to this project or a subordinate task.

Materialtyp:

If a type is selected here, only materials of that type are listed (material types are self-defined lookup values).

Lagerort:

If a location is selected here, only materials are listed which are stored at this location.

Verwendungsart:

Specify what type of material to list: Immediate, Consumable, and/or Time material.

Search from:

If a date is entered here, only materials are listed whose validity start date is either blank or on or before this date.

Search up to:

If a date is entered here, only materials are listed whose validity end date is either blank or on or after this date.


Search

Clicking on this button starts the search.




Material list

List of all materials in the company sorted by article number, which meet the filter criteria previously set in the selection mask.



Field Descriptions

Del.:

This field indicates whether the related field for this profile may be deleted.

Stat.:

Status of a stock item, i.e. it is warned in traffic light colors if there are too few or too many materials in the stock.

Int. Item Number:

Unique internal article number of a material.

Ext. Item Number:

Unique external number of an article, material, etc.

Material Name:

Name or description of a material.

Usage Category:

Usage type of a material, that is, for which purposes this material is used.

Instant materials are materials that are assigned to tasks once, whereby the corresponding costs and/or revenues are transferred directly to the project budget, e.g. computers, printers, etc.

Consumable materials are materials that can be consumed by a person on a unit basis (pieces, meters, etc.) on a daily basis, e.g. cables, sockets, etc.

Time materials are materials that can be consumed by a person on an hourly basis and to the day, analogous to time recording, e.g. company cars, rented rooms, etc.

Material Type:

Type of a material, that is, to which material group it belongs. The material types can be defined as lookup values themselves.

Task:

Task for which an immediate material was purchased or for which a consumption was entered for consumable or time materials.

Amount:

How much of a material was purchased or consumed.

Unit:

Unit of the material, e.g. piece, kg, m, etc. The units can be defined as lookup values themselves. For time materials, the unit is always "hrs".

Start Date:

The purchase date of an immediate material, or the validity start date of a consumption or time material. For material consumptions, this date is the same as the posting date.

End Date:

The purchase date of an immediate material, or the validity end date of a consumption or time material. For time material consumptions, the time span is displayed here.

Stock:

Indicates whether the consumable is a stock item (= ) or not (= ) .

Stock Loc.:

Location where a stock consumable is stored.

Text:

Text field

Variable text field that can be customized programmatically.

Stock Min.:

Number of minimum available consumables in the warehouse, i.e. this is how many materials should always be in stock in the warehouse.
This value is mainly used to check the stock in order to see immediately what should be reordered.

Stock Max.:

Maximum number of consumables available in the warehouse, i.e. the maximum number of materials that can or should be in stock in the warehouse.
This value is mainly used as a control upper limit for reordering materials.

Text:

Text field

Variable text field that can be customized programmatically.

Stock Ent.:

Total number of receipts of consumables into the warehouse, that is, the total number of materials by which the warehouse stock was increased.

Text:

Text field

Variable text field that can be customized programmatically.

Stock Rem.:

The total number of issues of consumables from the warehouse, that is, the total number of materials by which the warehouse stock was reduced.

Text:

Text field

Variable text field that can be customized programmatically.

Stock Act.:

Number of total consumables currently available in the warehouse. This value corresponds to the difference between the total inflow (= material increase) minus the total outflow (= material reduction).

Text:

Text field

Variable text field that can be customized programmatically.

Stock Used:

Total number of consumables already consumed from the warehouse, i.e. materials withdrawn from the warehouse. This value is determined from the total number of day-specific entries recorded for material consumption.

Planned-Actual-Comp.:

Calculation formula:
Target-actual comparison= Best. Ver. (= number of total stock materials already consumed)
Best. Act. (= number of total currently available stock materials)

Plnd.Cost/Unit:

Planned costs of an immediate material per unit, that is, these costs are acquisition costs of an ordered but not yet paid immediate material.

(For consumables or time materials, this value is always 0.00)

Act. Cost/Unit:

Purchase price (EK) per unit, which have an impact on the internal budget (= costs of the company).

Sale Price/Unit:

Sales price (VK) per unit, which has an impact on the external budget (= revenue of the company, which can be settled with a client).

Plnd. Cost/Tot.:

Planned total costs of an immediate material of all units, that is, these costs are acquisition costs of all ordered but not yet paid immediate materials.

Formula:
Planned total cost = Planned cost per unit * Quantity

(For consumable or time materials, this value is always 0.00)

Purch Cost/Tot.:

Total purchase prices (EKs) of all units, which have an impact on the internal budget (= costs of the company).

Formula:
Total purchase price = purchase price per unit * quantity

Sale Price/Tot.:

Total sales prices (VKs) of all units, which have an impact on the external budget (= revenue of the company, which can be settled with a client).

Formula:
Total sales price = sales price per unit * quantity

U-Margin:

Profit (= ) or loss value (= ) as the difference between the total sales price and the total purchase price.

Formula:
Consumption margin = total sales price - total purchase price.

Invoice No:

Display each related invoice with status, invoice number and invoice date.

The following statuses are displayed:
Task is [Not billable], or capture is marked as [Not billable].
Task is [Fixed price task].
Billable, but not yet billed.
Already billed or Already (indirectly) billed by billed fixed price task.

For consumables, a progress bar is also displayed to represent whether and how much has already been billed.

MT (Open):

Externally still open, that is, not yet settled (actual) net costs of billable materials.

MT (Invo.):

Externally already billed (actual) net cost of billable materials.

I-Margin:

Profit (= ) or loss value (= ) as difference between the externally already settled net costs and the total purchase price of the materials.

Formula:
Invoice margin = Billed net costs - total purchase price.

Act.:

Action number: 1 - display, 2 - edit, 3 - create, 4 - delete
