Reporting > Projects > Project Personnel Deployment Plan

Graphical display of all employees in a project with planned hours for a specific timespan


This diagram shows the use in working hours of all employees within a project.

Depending on the presence of an employee and depending on the required deployment, the total number of working hours of all employees per day can vary.




Criteria selection

Criteria selection for the display in the project-staff-schedule

In the selection mask you have to select a project for which you want to see the efforts of the single employees in which time period (time period from - time period to).

If "Time from project" is ticked, the [Time span from] and [Time span to] are taken over with the corresponding values from the selected project.

Automatic", "Days", "Weeks", "Months" or "Years" can be set as the display type. If "automatic" is selected, the most suitable display type for the diagram size is set. This selection value is responsible for the division of the X-axis.

If "Add up hours" is ticked, all hours of the employee per time unit will be displayed added up, otherwise one entry per employee will be displayed next to each other.



Field Descriptions

Time Span from:

This field displays the point in time from which data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Person:

This selection field is used to select the person whose data is to be edited or displayed depending on the action.

Project:

The Project field lists the tasks (projects or tasks) for which you have the appropriate rights and the corresponding filter criteria apply.

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Cal. Display:

Used to specify the calendar time intervals per column, i.e. days, weeks, months or years.

If "Automatic" has been set, the calendar time interval is determined programmatically based on the length of the total period to be displayed.

Level:

The number you enter in this field controls how many project levels are displayed in the report.

Sublevel "0" corresponds to the highest level.

Time from Project:

If this checkbox is checked, the entered time span is automatically overwritten with the time data from the selected project.

Add Hours:

If this checkbox is ticked, all hours of the employee per time unit will be displayed added up, otherwise one entry per project will be displayed diagonally one behind the other.

Show Unknown:

If this checkbox is checked, "unknown" tasks will also be displayed.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Diagram

Diagram Explanation of the project-staff-schedule
A vertical red line is drawn in front of today's date in the diagram. Everything to the left of the line corresponds to the actual data, everything to the right of the line corresponds to the planned target data.
On the y-axis (high value) the working time (= potential) is displayed in hours (h), on the x-axis (right value) the calendar time.
For the representation approx. 10 colors are given. If more than 10 colors are needed then the colors are repeated with a slight change in brightness.

NOTE:
Attention: With many persons the color can become very fast no more distinguishable. Please limit in such a case the displayed data by means of filters, period, etc.
.
