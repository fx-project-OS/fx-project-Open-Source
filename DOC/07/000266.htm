Administration > Client > Scheduler

Definition of times for certain special programs executed in the background


Settings for the scheduler, i.e. for time-controlled fx-project programs such as:

  • ODBC synchronization
  • Time entry check
  • Email scheduling
  • Warnings
  • Project check
.
This allows you to specify which days of the week each program should run, as well as what times.




Adjusting scheduler settings

In this mask, the scheduling behavior can be adjusted for various available programs.

In the field Interval repetition intervals can be specified in the unit "minutes". If the field is filled with "0", the program is executed only once. The specification of an interval basically only makes sense for the e-mail program.

Depending on your ideas, you can have each of these programs not run at all, once or several times a week, daily or even several times a day.

A more sensible execution schedule would be, for example, the following:

Time tracking check: every workday, between 2:00 a.m. and 3:00 a.m., with no interval.

Email scheduling: every weekday, between 0:00 and 24:00, with an interval between two and ten minutes.

ODBC synchronization: every weekday, between 22:00 and 23:00, with no interval.

Alerts: every weekday, between 3:00 and 4:00 am, with no interval.

Projects check: every weekday, between 4:00 and 5:00 am, without interval.

NOTE:
Note: All programs listed here must be activated beforehand under "Administration -> License Points -> Special Programs" in order to be able to schedule them.



Field Descriptions

Only Cust. Data:

Activate or deactivate an entry by placing a tick in the "checkbox".

Program Name:

Names the scheduler program.

Mo:

This field can be used to check whether the resource is available on Monday or not.

Tu:

In this field you can check if the resource is available on Tuesday or not.

We:

In this field you can check if the resource is available on Wednesday or not.

Th:

In this field you can check if the resource is available on Thursday or not.

Fr:

In this field you can check whether the resource is available on Friday or not.

Sa:

In this field you can check if the resource is available on Saturday (weekend) or not.

Su:

In this field you can check if the resource is available on Sunday (weekend) or not.

Pland From:

Specifies the target start time.

Pland To:

Specifies the target end time.

Int:

Specifies the interval at which the scheduler program is to be restarted.

Text:

Time unit min. = minutes


Multi Function

Clicking this button will make the changes.
