Creation > Employees > Annual Vacation

Management of annual leave and/or carry-over of previous year's leave


The annual leave of each internal employee can be changed here comfortably, as well as a carry-over value from the previous year can be set.




Person selection

Define selection criteria to filter the persons to be listed for which annual leave is to be displayed and/or defined.



Field Descriptions

Search Text:

This field is used to enter the string to search for.

Time Span from:

This field displays the point in time from which data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Active Employee:

This checkbox can be used to select whether all employees who match the selection criteria should be displayed or only those employees who have a displayable active record in this period.

Checkbox ticked = Only employees who have a displayable active record.

Department:

Indicates the department in which the employee works.

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Display History:

If this checkbox is ticked, an additional history is displayed in addition to the normal data.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Previous year's holiday

Previous year's holiday

Enter the leave from the previous year if the person's additional data has already been entered.

You see all employees of the client with the annual leave entitlement, the current remaining leave, as well as the input field "from previous year".

Enter the person's leave from the previous year in the field "from previous year" and repeat this process until all persons who still have leave from the previous year have been taken into account.

Then click on OK. The remaining leave is now corrected for each person.



Field Descriptions

Year:

The year is specified in this field.

Vacation per Year:

Shows the employee's vacation per year (in days).

Carry from Prv. Year:

Vacation carried over from the previous year.

Vac. Taken:

Display field for vacation taken in the current year.

Active:

Display whether a special program is active or inactive.


Year

Clicking this button will make the changes.
