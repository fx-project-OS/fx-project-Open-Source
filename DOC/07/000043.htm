Planning > Persons > Assign Person Resources (HR)

Assignment of person resources (HR) to tasks with various evaluation options


The resource planning of persons is used to display as well as to assign resources. Basically, persons are assigned to tasks, because time recording can only be done on tasks. If a person is assigned to a project or subproject, this person is also automatically assigned to all underlying tasks. When selecting tasks, multiple selection is possible, which is achieved by holding down the CTRL key. Persons with certain skills can be filtered for display using the "Skill" selection field. If the checkmark is set at "Show projects", already assigned projects of the individual persons are graphically displayed.




Employee utilization / employee availability

Settings for the graphical display of employee workload/availability

Only employees with the ability Employee role -> "Project manager" or "General manager" may carry out resource planning.

Resource planning is only possible if a project or task has been selected.

The main client 1 can additionally manage its resources across clients, if this option was selected during the installation.



Field Descriptions

Project:

The Project field lists the tasks (projects or tasks) for which you have the appropriate rights and the corresponding filter criteria apply.

Person:

In the selection field Person all employees defined in the person criteria are listed. All persons contained here are displayed in the graphic formatting.

If only one employee is selected, only changes can be made to this person.

Dept.:

Indicates the department in which the employee works.

Skill:

By selection of a certain ability or an ability group the display of the indicated coworkers can be influenced, since then only changes can be accomplished with the persons, who have the appropriate abilities. All other employees will be displayed as well, but only "greyed out".

Time from:

A date must be entered in this field from which time the graphical representation of the employee workload should begin.
(The date must be entered in the format defined in the user settings.)

Time to:

A date must be entered in this field by which time the graphical representation of the employee workload should occur.
(The date must be entered in the format defined in the user settings.)

Calendar Displ. Mode:

Used to specify the calendar time intervals per column, i.e. days, weeks, months or years.

If "Automatic" has been set, the calendar time interval is determined programmatically based on the length of the total period to be displayed.

Time from Project:

If this checkbox is checked, the entered time span is automatically overwritten with the time data from the selected project.

Display Utilization:

If this checkbox is ticked, the employee workload is displayed in the graphical representation of the employee data.

Disp. active tasks:

Determines whether active tasks should be displayed or not.

If this checkbox is activated, all tasks with the status "active" are listed. Otherwise, these tasks will be hidden.
This checkbox should remain checked in the resource planning always to get correct utilization values!

Disp. service.:

Determines whether maintenance tasks should be displayed or not.

A maintenance task is a task that does not cause any workload because, for example, the effort was planned with 0. Useful, for example, for recurring long-running tasks without exactly predictable scheduling.

Disp. Only Planned:

Used to display only planned-target data, i.e. actual data from time recording, appointments, etc. are not taken into account.

Display Projects:

If this checkbox is selected, the project workloads of the individual projects to which the employee is already assigned are displayed in the graphical representation of the employee data.

Disp. approved tasks:

Determines whether or not to display scheduled tasks with approved budgets.

If this checkbox is activated, all tasks with the statuses "planned" and with approved person budgets are listed. Otherwise these tasks will be hidden.

Only Assigned Pers.:

If this mode is activated, only persons are listed who are already assigned to at least one of the selected tasks.

Disp. planned tasks:

Determines whether or not to display scheduled tasks without approved budgets.

If this checkbox is activated, all tasks with the statuses "planned" and without approved person budget are listed. Otherwise these tasks will be hidden.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.
