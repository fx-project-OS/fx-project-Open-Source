Reference selection

Reference for rights assignment

Here, a reference category (projects, documents or reports) must first be selected and then a subcategory for this reference.

If "Rights assignment" is called up from another program function, these values are already preset and are only displayed.

Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Manage access rights

Manage access rights

In this screen, access rights can be assigned to specific departments, user groups or individual users.
If the user already has a direct right to selected references (through his assignment to these or superordinate project), this is displayed with a blue background.

The access rights for documents are assigned to actions and mean in detail:


NOTE:
All: All access rights are granted.

retrieve: The user is allowed to download the document.

check in/out: The user may download or upload the document for
Down- or upload the document for editing. (In the meantime, the file may then only be downloaded.

archive: The file will be archived and will not show up in future searches.

delete: The file will be deleted.
