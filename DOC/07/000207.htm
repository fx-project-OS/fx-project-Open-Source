Reporting > Times > Skill Statistics

Graphical processing of time data grouped by skills


Statistical evaluation of the skill distribution to each individual task or project. Here, the selection criteria can be conveniently adjusted to obtain either a global project listing or an individual listing for a specific employee in a specific time period.




Display criteria for project statistics

In the mask Display criteria for project statistics the selection is made which statistical evaluation of the capability distribution is to be listed.

At least in one of the fields Project, Person, 1st Project Leader or Skill a specified entry must be selected, which serves as a basis for the determination of the statistical data. A selection of more than one entry limits the amount of data determined accordingly.

The period to be displayed can be specified via the two input fields Time Span from and Time Span to. If no entry is made, the entire relevant period is included in the calculation.

By ticking the checkbox Complete the determined skills for each relevant task are displayed in detail.

The desired data are determined after a click on the button Display and displayed in the mask Project statistics prepared. There it can be processed further via the [File Manager].



Field Descriptions

Project:

The Project field lists the tasks (projects or tasks) for which you have the appropriate rights and the corresponding filter criteria apply.

Person:

This selection field is used to select the person whose data is to be edited or displayed depending on the action.

Time Span from:

This field displays the point in time from which data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

1st Project Leader:

In this field the manager of a project is displayed or must be selected from a selection list. The project manager is defined once in the master project, and is then preselected as a PROPOSAL in all underlying projects.

Green displayed persons in the selection field are project leaders, which are already assigned to at least one project in this role.

Project managers are created using the role mask with role "Project Manager".

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Skill:

Selecting a specific skill or skill group to group employees.

Complete:

Ticking this checkbox will display the identified skills in detail for each relevant task.


Display




Project statistics

Display detailed information of the skill distribution for a selected project/employee.

In the mask Project statistics all important information concerning the skill distribution for a selected project or employee is displayed clearly. The information content can vary depending on the display options set (see help for mask Display criteria for project statistics).
