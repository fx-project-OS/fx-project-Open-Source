Workflow menu (workflow slider)

The workflow menu, also called workflow slider, is located on the left side of the screen and is used to call up the program functions with visual support.

The workflow slider is the upper of the two slider elements on the left side.

Approaching with the mouse cursor, the subtle "tab" becomes larger and labeled. By clicking on this "tab", the workflow menu can be expanded and collapsed.

Opened, the user gets an overview of all main sections of authorized program functions.



Within the workflow menu, the user has the following control options:

Black and white display → Switch to color display.
color display → Switch to black and white display


By clicking on the respective menu items, you can navigate through to the desired program functions.



The following options may be available for the respective action:

Invoke program function in mode: show.
call program function in mode: new creation
Call program function in mode: Modify
Call program function in mode: delete
