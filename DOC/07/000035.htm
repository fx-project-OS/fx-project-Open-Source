Administration > Profile/User > Profile Names

Manage profile names to which program functions with access rights can be assigned


Several program functions (program parts) can be grouped together in the form of profiles and then assigned to the user. The profile names required for this can be assigned in this program function.




User profile groups

Create, activate and deactivate user profile groups.

In User profile groups all activated (tick) and deactivated (no tick) user groups are displayed.

In addition, an empty field is available to create a new user group.

NOTE:
The entries for the field "groupname" can be administrated at the menue item"Administration/Values in look-up table".
In case you have no right on this programm resp. it is not shown in your main menue, then please contact your administrator, project manager or supervisor.



Field Descriptions

Only Cust. Data:

In Checkbox is indicated, which user group is active (hook) or inactive (no hook).

Group Name:

Group Name contains the names of the user groups.

No.:

Number of entries


Save

By clicking on this button, the changes are made. The text of the button depends on the current action mode.
