Reporting > Persons > Communication Report

Evaluation of communication data of persons


The person communication report is a detailed report about the most different communication data of persons. The freely definable communication type can be set as well as a search text, postcode, country code or person type to limit the data found. The integrated assignment of rights ensures that each person is only listed the data intended for them.




Determine selection criteria

Here you can define the selection criteria for the person communication report.



Field Descriptions

Communication Type:

This field is used to select the type of communication connection, such as telephone, fax, e-mail, etc., from a drop-down menu.

Allocated to Object:

This field contains the "parent unit".
For projects, this would be the parent project [(sub)project].

For documents this would be the unit you belong to (e.g. project, appointment, note etc.).

Search Text:

This field is used to enter the string to search for.

Country Code:

This field is used to select a country code from a country table.

The country code corresponds to the currently valid car license plates, e.g. "USA", "F", "NL", "D" etc.

Display Annotation:

Here you can define whether the annotation should be displayed or not. Checking this box means that the annotation column should be displayed in the report, otherwise it will be hidden.

Inactive:

By setting the checkbox, inactive persons (employees, clients, etc.) are also displayed.

Type of Person:

Categorization of persons, e.g. clients, employees, customers etc. are to be distinguished.

Sort Sequence:

Specify the sort order of the displayed fields.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Report

Report of the selected communication data.



Field Descriptions

Number:

The actual communication connection, such as telephone number, fax number, e-mail address, etc., is entered or displayed in this field. The entries in this field depend on the previously selected communication type.

Annot.:

This field displays general comments about a project.

Name:

In this field the family name of the person or the name of the legal entity (e.g. company name) is displayed or entered.

Address:

This is an internal key for the unique identification of addresses.

Contact:

This field shows the contact person or the customer. By selecting a contact person, in some cases (e.g. project definition) a customer is automatically selected as well, since each contact person is assigned to exactly one customer.

A contact person is created via the input mask in the menu item "Master data".

Number:

Specifies the identification number of the communication entry.

Editor:

Indicates which user last edited a record.
