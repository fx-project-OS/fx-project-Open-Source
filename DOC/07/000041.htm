Execution > Appointments > Calendar

Appointment calendar in graphical form


Listing of a person's current appointments in a configurable calendar format (by the day, week or month).




Date selection

Screen for appointment selection

In the top line of the calendar control you have the possibility to select which entries you would like to see in your calendar.

The select field Employee Appointment only appears if the user has the ability "Employee role: Business manager or Project manager" and also the setting "Management and project managers see all appointments" is set to "Yes". If this is the case, then the user can see appointments of other persons by selecting their name in the select field (or "All" for the appointments of all persons).

There are different views to choose from, e.g. one-day view or seven-day view. With a click on Display the calendar is created according to the new settings.

With a click on Today you get to today's date

By entering the calendar week you can, as an alternative to the date, switch to a desired date.



Field Descriptions

Date Entry:

With the help of the field "Date specification" you can directly display a desired day.

This option is preferable if you want to "jump" over a longer period of time, because here only one date has to be entered instead of a tedious scrolling forward or backward.

CW:

The input of the calendar week is an alternative to the date, with which you can jump to the specified calendar week, followed by the click on the button "Show".

View:

In the field "View" you can specify how many days should be displayed on the calendar, e.g. one-day view or 7-day week etc.

Employee Appointment:

In this field you can specify whether (an authorized person) wants to see the appointments of his colleagues.

In the case that an employee wants to see the appointments of another employee, the respective employee has to be selected and "Show" has to be clicked. It is also possible to see the appointments of all employees at the same time via the entry "all".

Projects:

By ticking, projects are displayed in the appointment calendar. If a project is found in the displayed period (i.e. if a start or end date of a (sub-)project falls into the selected period, you can examine this project in the project view by clicking on the project symbol.

Task:

Tasks are displayed in the appointment calendar by ticking them. If a task is found in the displayed period (i.e. a start or end date of a task falls into the selected period), you can examine this task in the project view by clicking on the task symbol.

Business:

Business appointments are displayed in the appointment calendar by checking them off.

Private:

By unchecking the box, private appointments are displayed in the appointment calendar.

Leave:

Field to display an employee's vacation days.

Training:

Field to display the training days of an employee.

Sick:

Field to display the sick days of an employee.

Flextime:

This field indicates whether the worker is entitled to work in floating day.

If the user is authorized to do so, he can book time records to "Floating day" within the time recording.

This allows the user to enter a floating day for e.g. Friday, if he has already fulfilled his target until Thursday.

By making this entry, the user is also prevented from being registered via the time recording check program, even though he has already worked the necessary hours.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.

Today

Button of the date selection. After clicking you will find yourself at today's date.
