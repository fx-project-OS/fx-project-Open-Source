Toolbar

The toolbar of fx-project is located at the top of the screen and contains the most important control options for fx-project in addition to the fx-project logo and a search function.



Control Center

Calls the program function Control>General>Control Center. This shows an overview of all current actions, i.e. assigned tasks, appointments and other announcements or messages.


Back Button

The "Back Button" calls the program function which was last executed before the current program function.


Favorites

Using the popup for "Favorites", the last 20 (Last 20) or most frequent 20 (Top 20) program functions are listed and can be accessed with a click. This allows a quick change to the desired and frequently used program functions.


Filter / Portfolio Manager

The "Filter" can be used to reduce the amount of data for the individual program functions, e.g. for projects the status or project owner etc. can be restricted for all displayed project parts. In this way, reports on a large amount of data are restricted in a controlled manner and shortened accordingly. The filter combination can be saved under a filter name. This makes it easy to create and retrieve multiple project portfolios.


Settings

The popup for "Settings" allows additional and/or optional display options to be enabled for the current program function.


Help

The popup for "Help" displays detailed help documentation for the current program function.


Info

Calls the program function Administration>Help>Info. This displays information regarding this fx-project version such as version number, patch, database, browser and logged in users etc.


Logout

To optionally exit the program (= logout) click on the logout button in the upper right corner.


Search field

The "search field" at the top right allows a quick entry into the desired program function. It is sufficient to enter a part of the search term, e.g. "travel", to display all program functions in which travel plays a role. The prerequisite for calling up the desired program functions is, of course, the authorization for the respective programs, which can be activated and assigned with Administration>Profile/User>Profile Program Assignment.
