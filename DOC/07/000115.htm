Administration > Client > Holidays

Manage client-wide holidays


The corresponding holidays can be activated separately for each client. Holidays can be a full or half working day, which have an effect on the planned working time of the employees for time recording etc.. Additional, not recorded holidays can be defined.




Holidays for the year / and for the country

Here all holidays can be changed, deleted or created completely new.

If you create (half) bridge days, (half) holidays are created for the entire staff.

However, the prerequisite for this is that

1. the employee has not yet entered a leave or sick day on this day
2. the employee has entered a workday on this day
3. the employee has already entered a defined start and duration of work on this day
4. the bridge day is in the future.

For each (half) bridge day created, each employee is credited with one (half) day of leave.



Field Descriptions

Chk.:

Activate or deactivate an entry by placing a tick in the "checkbox".

Day:

Full day of the week name.

Date:

Used to enter or display a date of the day. If a date entry is required, the default value (today's date) is usually already preset.

Holiday:

Specifies the holiday name.

Half:

Indicates whether this holiday is full-day, half-day, or display-only.


Chk.

Clicking this button will make the changes.
