Milestones

Listing of all active milestones and all planned but not yet active milestones until the end of the next month.



Field Descriptions

Status:

Status of the milestone in traffic light graphics incl. warning level from 0-10.

Date:

Target date of the milestone.

Milestone:

Name of the milestone.

Type:

Milestone type, i.e.:
S = Dynamic milestone depending on the start date of the assigned project.
E = Dynamic milestone depending on the end date of the assigned project.
F = Fixed milestone independent of dates of the assigned project.
(T) = Number of days difference from assigned project date.

Allocated to Object:

Project to which this milestone is assigned. For dynamic milestones, the milestone date automatically moves with the assigned project.

period:

Scheduled start and end date of the milestone's assigned project.
