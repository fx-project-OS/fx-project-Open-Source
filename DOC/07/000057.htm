Reporting > Times > Time Data Report

Listing of the time recording of individual employees with display of the target/actual times and absence days


Authorized persons can view the hourly reports of individual/all employees of any period for statistical purposes. The level of detail can be varied using the display type.




Hourly report selection

Hourly report selection

Specify selection criteria according to which the hour report for individual employees is to be formatted.

In particular, the time span to be displayed can be set.
If the display type is "Automatic", the program selects the time span to be displayed automatically.



Field Descriptions

Person:

This selection field is used to select the person whose data is to be edited or displayed depending on the action.

Display:

Specifies how certain data should be formatted within a report.

Time Span from:

This field displays the point in time from which data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Text:

Text field

Variable text field that can be customized programmatically.

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Empty Entries:

By checking or unchecking this field, you can determine whether irrelevant entries (=blank entries) are to be displayed or not.

If, for example, a month is listed for a project, but only two days in this month were worked for this project, you can decide here whether the entire month or only these two days should be displayed.

Hours:

Number of hours according to context, e.g. number of overtime hours to be paid, etc.

Estimat.:

Determine whether or not plan estimates, that is, days with assigned plan tasks, should be displayed in the future.

Workday:

Indicates how many days the employee is required to work per week.

Leave:

Field to display an employee's vacation days.

Spec. Leave:

Field to display an employee's special leave days.

Training:

Field to display the training days of an employee.

Sick:

Field for the display of the sick days of an employee.

Absent:

Field to display the days of absence of an employee.

Flextime:

This field indicates whether the worker is entitled to work in floating day.

If the user is authorized to do so, he can book time records to "Floating day" within the time recording.

This allows the user to enter a floating day for e.g. Friday, if he has already fulfilled his target until Thursday.

By making this entry, the user is also prevented from being registered via the time recording check program, even though he has already worked the necessary hours.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Hourly Report

Hourly Report

The hour report for the individual employees is displayed here, depending on the time period set.



Field Descriptions

Days:

Full day of the week name.

Time:

Used to enter or display a time.

Date:

Used to enter or display a date of the day. If a date entry is required, the default value (today's date) is usually already preset.

Workday:

Indicates how many days the employee is required to work per week.

Leave:

Field to display an employee's vacation days.

Sp. Leave:

Field to display an employee's special leave days.

Training:

Field to show how many days of training an employee has.

Sick:

Here is the sum of the days in which the employee was sick.

Absent:

Field to display the days of absence of an employee.

Flextime:

Number of floating days in this month.

Plan.Hrs.:

Indicates how many hours an employee must work in a given period of time

Act.Hrs.:

Indicates how many hours an employee actually worked in a given time period.

Delta:

The delta hours are the difference between actual and target hours. They indicate how many hours the person has worked more or less than contractually agreed in the specified period (= overtime or minus hours).
