General procedure for defining a project

In order to be able to create a project, some prerequisites have to be met so that the mandatory fields can be filled in during input:


Create client:

When defining the project, it is important that the principals have already been entered in the Creation>Customers>Basic Data program function. In addition, the contact person must be defined in Creation>Customers>Contact Person. Only if these requirements are met, a project can also be assigned to a principal.


Employee data:

In order to be able to plan a project completely, all employees involved in the project should be created in the program functions Creation>Employees>Basic Data or Creation>Contractors>Basic Data beforehand. Likewise, the settings regarding working time and availability should be correctly stored in the respective additional data.
The resource planning is then done either in the function Planning>Persons>Assign Person Resources (HR) or most simply directly in the graphical project planning Planning>Projects>Matrix: Graphically as Gantt.


Determine project manager, deputy project manager and project manager:

In order to select a project manager, the appropriate employee roles must be assigned to the desired individuals. To do this, first go to the program function Creation>Employees>Roles (for internal persons) or Creation>Contractors>Roles (for external persons) and assign a special "employee role" to the corresponding employee, in this case project manager or project manager.


In addition, special skills of the employees can be stored, so that employees with special knowledge can be found more quickly during resource planning. These skills are also stored in the project, in order to be able to make later appropriate evaluations on projects. In Administration>Skills>Skill Groups and Administration>Skills>Skills, the characteristics of these skills can be defined individually as a group and subgroup.


Creating a project:

Once the above requirements are met, the desired project can be defined in Creation>Projects>Create Main Project. In the second step, using the graphical input in Planning>Projects>Matrix: Graphically as Gantt, the project can be designed in the matrix, the effort can be planned, The project details can be set and the resources can be assigned.
