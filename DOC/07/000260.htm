Planning > Costs > Budget Approval

Approve Task Budgets (if Budget Approval is activated client-wide)


Listing (optionally also with history) of already approved task budgets for persons or material.

In the new creation or change mode, planned budgets can be approved, equipped with new additional budget or optionally provided with an approval text. Furthermore, the planned deadline period can also be confirmed or changed here, so that time entries are only possible in this area.


NOTE:
Only the additional budget, i.e. the remaining amount between the planned and already approved budget, needs to be re-approved. The sum of all approved additional budgets results in the total approved budget.




Selection of tasks to be approved

Selection of tasks to be approved

Here you can define which tasks should be displayed for budget approval.

If a parent project or subproject is selected, all subordinate tasks of this project are listed.

If a time period is specified, only tasks that lie within this period are displayed.

If History is checked, then for each displayed task also already existing previous approvals are displayed.

Only those budget types are displayed which correspond to the checked type, i.e. internal and/or external.

NOTE:
Only tasks are displayed, which are also visible in the selection mask, i.e. the general project filter settings are effective.



Field Descriptions

Project:

The Project field lists the tasks (projects or tasks) for which you have the appropriate rights and the corresponding filter criteria apply.

Start Date:

If a date is written into this field, then all determined dates are limited to dates whose start date is after the date entered here.

End Date:

If a date is written into this field, then all determined dates are limited to dates whose end date is before the date entered here.

Show History:

If this checkbox is ticked, an additional history is displayed in addition to the normal data.

Init. Time Span:

Specifies whether the new approval period should be preset with the planned target period of the task.

Int. Total Budget:

This value specifies the total net internal budget (target), that is, this is the sum of the internal personnel budget + internal material budget.

Ext. TBudget:

This value specifies the total external net budget (target), that is, this is the sum of the external personnel budget + external material budget.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




List of tasks to be approved

List of tasks to be approved

Here all projects are listed according to the previously selected structure.

The individual lines are divided into internal and external budget data, both the planned and the already approved data are displayed. For tasks, the newly approved additional budget can be entered, as well as the newly approved period. The new additional budget to be approved usually corresponds to the difference between the planned budget and the total of the budgets already approved. This value is also the default value for the fields.

For tasks, a comment can also be added to each new approval.

If the checkbox is selected for projects, all subordinate tasks are automatically approved with the entered values. If no value is entered, the difference between planned and already approved budget is automatically used for new approval.

NOTE:
If History was checked, then for each task also possibly existing old budget approvals are listed in detail.



Field Descriptions

Only Cust. Data:

Activate or deactivate an entry by placing a tick in the "checkbox".

Proj. No.:

This is the directory number for a project.

The directory number cannot be changed for parent projects.

The directory number is automatically adjusted when deleting or moving.

By manually changing the directory number, you can move projects within a hierarchy.

Status:

This field shows how far a project has already progressed. In input mode, you can assign an attribute from a selection list to a project here, which reflects whether or how far a project is already advanced.

The status can also be used to decide whether project data should be included in a report or not.

Text:

Budget: Int. planned, Int. approved, Int. approval change

Person Budget:

This value indicates the general net budget (target) for persons, i.e. both the internal and the external budget.

Material Budget:

This value specifies the overall net (target) budget for materials, that is, both the internal and external budget.

Total Budget:

This value indicates the overall total net budget (target) for people and materials, i.e. both the internal and external budget.

Pln. Start:

The target start date of the project is displayed or specified here, i.e. from when the project is to be started according to planning.

Pln. End:

The target end date of the respective reference object, e.g. project, time data, trip data, etc. is displayed or specified here, i.e. by when the object should be completed according to planning.


Multi Function

Clicking this button will make the changes.

Report of the approved tasks
