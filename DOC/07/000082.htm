Planning > Persons > Search

Search and list of specific persons


Persons are searched and found according to freely defined patterns. These patterns can be composed of name, person type, job title or department.




Search people

Search people

Here you can enter the search criteria for the person search. The more detailed the entries are, the more accurate the hit list will be.

NOTE:
If all fields remain empty, then you will receive all persons included in the system, which, however, depending on the respective amount of data, could lead to longer waiting times.



Field Descriptions

Name:

In this field the family name of the person or the name of the legal entity (e.g. company name) is displayed or entered.

Type of Person:

Categorization of persons, e.g. clients, employees, customers etc. are to be distinguished.

First Name:

This field displays or enters the first name of a person if the field is empty (maximum 40 characters allowed).

Job Description:

In this field the position of a person in a company according to the job description is selected, e.g. "management", "administration", "project manager" etc.. This field therefore only contains the job position under which a person has been hired.

However, each person can also be assigned different employee roles in the corresponding skills mask, e.g. a "business manager" can also or in particular receive the skill "project manager" and then be used as such in a project accordingly.

So that a project can be created, at least one person must be assigned the employee role "project manager".

Contact Possibility:

By activating this checkbox, the search is extended to the communication data of the persons found. Otherwise, only personal data, such as name or position in the company, and address data are displayed.

Primary key in the "Department" table:

Indicates the department in which the employee works.


Search

Clicking on this button starts the search.




Hits of the people search

Hits of the people search

This list contains all persons matching the search criteria. In addition, their most important data is displayed.
By ticking the checkbox Contact Possibility you receive additional communication data for the persons found.

Name:

First Name:

This field displays or enters the first name of a person if the field is empty (maximum 40 characters allowed).

PersonType:

Categorization of persons, e.g. clients, employees, customers etc. are to be distinguished.

Dept.:

Indicates the department in which the employee works.

Job Desc.:

In this field the position of a person in a company according to the job description is selected, e.g. "management", "administration", "project manager" etc.. This field therefore only contains the job position under which a person has been hired.

However, each person can also be assigned different employee roles in the corresponding skills mask, e.g. a "business manager" can also or in particular receive the skill "project manager" and then be used as such in a project accordingly.

Street:

The street with house number is entered in this field.

Code:

The postal code of the town is entered in this field.

Place:

The person's place of residence is entered in this field.

Cntry:

This field is used to select a country code from a country table.

The country code corresponds to the currently valid car license plates, e.g. "USA", "F", "NL", "D" etc.

Communication Type:

This field is used to select the type of communication connection, such as telephone, fax, e-mail, etc., from a drop-down menu.

Number:

The actual communication connection, such as telephone number, fax number, e-mail address, etc., is entered or displayed in this field. The entries in this field depend on the previously selected communication type.
