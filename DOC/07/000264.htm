Display Type:

Selection field for selecting in which time display type the report is to be displayed. Monthly, quarterly or yearly.

Show Empty Entries:

By checking or unchecking this field, you can determine whether irrelevant entries (=blank entries) are to be displayed or not.

If, for example, a month is listed for a project, but only two days in this month were worked for this project, you can decide here whether the entire month or only these two days should be displayed.

Inter.-Report:

If set, interim reports are displayed according to the display type set, i.e. weekly reports if displayed by the day, monthly reports if displayed by the week, quarterly reports if displayed by the month and annual reports if displayed by the quarter.

Glob.-Report:

If set, a full report is displayed provided that there is more than one entry in the report.

Use Filter:

Specifies whether the global filter settings should be used or not.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.

Field Descriptions

Cost Center:

Unique assignment number to a cost center.

Cost Type:

This field indicates whether the selected field is a cost field or not. If the field is not a cost field, it will always be displayed, if the field is a cost field of cost type "Project Manager or "General Manager", the user must have these additional permissions (Employee-Role (spec. Access Rights)) to see this field.

Service Type:

The activity type in cost accounting is a reference value that indicates the qualitative technical description of the production activity. Here, it can be classified between productive (=value-adding) and non-productive types.

Plan.Hrs.:

Indicates how many hours an employee must work in a given period of time

Act.Hrs.:

Indicates how many hours an employee actually worked in a given time period.

Pl.Cost/U:

Planned cost of an instant material per unit, that is, this cost is an estimate of the acquisition cost of the instant material.

(For consumable or time materials, this value is always 0.00)

Act.Cost/U:

Purchase price (EK) per unit, which have an impact on the internal budget (= costs of the company).

Delta:

The delta hours are the difference between actual and target hours. They indicate how many hours the person has worked more or less than contractually agreed in the specified period (= overtime or minus hours).
