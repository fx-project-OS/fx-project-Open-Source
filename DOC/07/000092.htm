Graphical overview (depending on access rights) of all employees about their vacation, training, sick days and/or actual hours


Graphical overview of all employees about their vacation/training days or sick days.




Graphical overview of absence days and/or actual hours worked

This graphical report gives you an insight into which employees have planned or taken leave and when, who is currently on a training course, or who has called in sick. You can see how many resources you have to do without in a certain period of time.

In both fields Time Span from and Time Span to you have to enter a date, i.e. for which period the graphical report should be created.

In the field Calendar Displ. Mode you can select how detailed the report should be prepared, i.e. whether you want to see days, weeks, months or years. In the "Automatic" mode, the program determines the display mode in such a way that all information still fits into the browser window.

By checking the Leave, Sick or Training boxes, you can determine which types of missing days should be displayed. Important! At least one entry must be checked.

In the selection field Department you can determine whether you want to see the graphical overview for all employees, or only for one specific department.

By checking the Active Employee field, the listing will only show employees who have days absent during the selected time period. If this field is unchecked, then all employees will be displayed.

Field Descriptions

From:

A date must be entered in this field from which time the graphical display of the USK plan should begin.
(The date must be entered in the format defined in the user settings.)

Cal. Display:

Used to specify the calendar time intervals per column, i.e. days, weeks, months or years.

If "Automatic" has been set, the calendar time interval is determined programmatically based on the length of the total period to be displayed.

Leave:

By ticking this checkbox the vacation days of the selected employees will be displayed graphically.

Training:

By ticking this checkbox, the training days of the selected employees will be displayed graphically.

Sick:

By ticking this checkbox, the sick days of the selected employees are displayed graphically.

Flextime:

This field indicates whether the worker is entitled to work in floating day.

If the user is authorized to do so, he can book time records to "Floating day" within the time recording.

This allows the user to enter a floating day for e.g. Friday, if he has already fulfilled his target until Thursday.

By making this entry, the user is also prevented from being registered via the time recording check program, even though he has already worked the necessary hours.

Act.Hrs.:

Indicates how many hours an employee actually worked in a given time period.

To:

A date must be entered in this field by which time the USK plan should be graphically displayed.
(The date must be entered in the format defined in the user settings.)

Department:

Here you can select the department and accordingly the employees belonging to the department for which you need the information. If no department is selected, then one receives the information to all coworkers.

Active Em.:

This checkbox can be used to select whether all employees who match the selection criteria should be displayed or only those employees who have a displayable active record in this period.

Checkbox ticked = Only employees who have a displayable active record.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.
