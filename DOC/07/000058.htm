Administration > Client > Reference Values

Individually extensible values for specific selection fields


Look-up values are user entries in the "Values table". These must be created at the beginning of the use of fx-project. The information stored is accessed by the individual programs depending on the type of look-up value. Example of form of address: the various forms of address Mr./Mrs. etc. can be stored for this.

Selection of lookup categories

Field Descriptions

Assignment to a field in table "Fields:

"Categorization" of the valuesTab. All tab values of the same type have the same ID_field (e.g. Mr, Mrs, Dr, Prof etc. for the field Salutation or German, English for the language).

Language:

The language set.

Short description of a reference:

This field displays the description of the reference name in short form.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Lookup values

All lookup values from the selected category are listed here and can either be changed, deleted or otherwise edited depending on the action set.

With the action "Create record" an empty line is inserted for the new lookup values.

With the action "Display", the values inserted by the user are displayed in bold.

With the action "Change" only the system values are displayed. These may not be changed or deleted.



Field Descriptions

Only Cust. Data:

Activate or deactivate an entry by placing a tick in the "checkbox".

Position:

Position of this value in the listing, i.e. where the entry should be placed.

Value:

This field represents a specific lookup value selected from a selection box (e.g. all country codes, currencies, etc.).

VAT:

VAT on the total price of certain items.


Save

Button to save data.
