Administration > Projects > Unblock Projects

Send request to unblock blocked projects that are currently being edited by another person


Listing of all currently blocked projects with the option to send a request to the person currently working on the project to unblock it.




List of all currently blocked projects

List of all currently blocked projects incl. all required information, who blocked which project and when. Additionally, the current status is displayed for each project, i.e. blocked, request sent, etc.

Furthermore, a request for unblocking can be sent for the corresponding project, if not already done.



Field Descriptions

Main Project:

Number and name of the project, which is currently edited by another person.

Project Manager:

Project manager of the project, which is currently processed by another person.

1st Project Leader:

Project manager of the project which is currently processed by another person.

In Use by:

Person who is currently editing this project.

Time:

Time since when this project is edited by another person today.

Status:

Time and status of the last executed action for this project.

Action Code:

Action button to send an "override request" or wait animation if a request has already been sent.
