2 F A
(2-FACTOR-AUTHENTICATION)  













VERSION 24.1  
  

Registration

If no device (PC / smartphone / tablet etc.) has yet been registered for 2FA, the following instructions will appear after logging in.


The QR code on your login page refers to the 2FA browser app website of your system. Now scan the QR code with your device and open this link in the browser of your device. The language, client number and user name are transmitted as parameters.


You must enter the registration code displayed in the browser app.


After clicking on the [Register] button, this is transmitted together with the unique "Device-ID" and stored for the respective user if it matches. A success message will then be displayed in the app and you will be automatically forwarded to the next point "Login".

Login

If 2FA is enabled for the client and a device (PC / smartphone / tablet etc.) is already stored for the user, the following instructions will appear after logging in or after successful registration.


The QR code on your login page refers to the 2FA browser app website of your system. Now scan the QR code with your device and open this link in the browser of your device. The language, client number and user name are transmitted as parameters.


A token code is now displayed in the browser app, which was/will be generated on the first call or after 60 seconds in synchronization with fx-project. The time bar at the top of the browser app shows how much time is left before a new token is generated.


This token code must be entered in the fx-project login screen and if it matches, the login is classified as "valid" and fx-project is loaded.


Information
