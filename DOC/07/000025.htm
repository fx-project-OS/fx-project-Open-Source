Administration > Users > User Settings

Miscellaneous settings that can be made by any user


Individual settings, which only apply to the own user.




Settings

Here, various settings can be changed according to personal preferences, e.g. the "Date format", the "Decimal separator", the "Number of decimal places", the "Number of list entries", the "Sort order of projects", etc.

Grayed settings have not yet been adjusted, i.e. these contain the specified original default values of your client.

User settings that have already been adjusted can be reset to the original default value by checking the "Delete" checkbox in front of the corresponding setting value.

A click on the "question mark" at the end of the line opens a separate help window for the corresponding setting value with an exact description of the change options.



Field Descriptions

Delete checkbox:

Checkbox to mark the already customized user setting which should be reset to the original default value.

Text:

Text field

Variable text field that can be customized programmatically.

Setting:

Short description of the setting. Extracted settings have not been changed yet, i.e. these still have the original default value.

Setting Value:

Set value of the setting. This value can be changed according to personal preferences.

questionmark:

A click on this icon opens a separate help window for the corresponding setting value with an exact description of the change possibilities.


Save

Clicking this button saves the settings.
