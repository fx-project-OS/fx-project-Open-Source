Administration > Profile/User > User Profile Assignment

Assign defined profiles with program functions to a user or vice versa


Any number of user profiles can be assigned to a user. Alternatively, it is possible to assign the corresponding users to a user profile. To assign a profile, the necessary license points of the profile must be activated on the points account. If a profile is withdrawn from a user, the assigned number of license points of the profile will be credited to the points account again. If the same program functions are available in different profiles (set of assigned program functions), the associated license points are only activated once as long as the program is available in one of the activated profiles.




Selection mask

Select users or user groups.

In Selection mask you select the desired user or user group, to which user groups or users are assigned in Assignment mask.

Users can only be assigned to active groups.

Field Descriptions

User Profile Groups:

In User Profile Groups you can select the user or group to which groups or users are to be assigned.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.

User Mode

Clicking on the User Mode button switches to user mode.

Group Mode

Clicking on the button Group Mode switches to group mode.




Assignment mask

Assign users to groups or groups to users.

In Assignment mask the desired user groups or users are assigned to the user or user group selected in Selection mask.

Users can only be assigned to active groups.
