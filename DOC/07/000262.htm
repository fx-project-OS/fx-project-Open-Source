Reporting > Persons > Personnel Project Deployment Plan

Graphical display of all assigned tasks of an employee with planned target efforts


This diagram shows the deployment of an employee (taking into account the contractual working time) in his various projects.
This illustrates that an employee can be involved in several projects at the same time.




Criteria selection

Criteria selection for the representation in the personnel project deployment plan

In the selection mask, the employee is selected for whom the workload is to be displayed in the corresponding time period (time period from - time period to).

The display type can be set to "automatic", "days", "weeks", "months" or "years". If "automatic" is selected, the most suitable display type for the chart size is set. This selection value is responsible for the division of the X-axis.

If "Add up hours" is checked, all hours of the employee per time unit will be displayed added up, otherwise one entry per task will be displayed next to each other.

If "none" level is entered, all tasks of this person in this period will be displayed, otherwise all subordinate tasks up to the specified level will be added up.



Field Descriptions

Time Span from:

This field displays the point in time from which data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Person:

This selection field is used to select the person whose data is to be edited or displayed depending on the action.

Project:

The Project field lists the tasks (projects or tasks) for which you have the appropriate rights and the corresponding filter criteria apply.

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Cal. Display:

Used to specify the calendar time intervals per column, i.e. days, weeks, months or years.

If "Automatic" has been set, the calendar time interval is determined programmatically based on the length of the total period to be displayed.

Level:

The number you enter in this field controls how many project levels are displayed in the report.

Sublevel "0" corresponds to the highest level.

Time from Project:

If this checkbox is checked, the entered time span is automatically overwritten with the time data from the selected project.

Add Hours:

If this checkbox is ticked, all hours of the employee per time unit will be displayed added up, otherwise one entry per project will be displayed diagonally one behind the other.

Show Unknown:

If this checkbox is checked, "unknown" tasks will also be displayed.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Diagram

Diagram Explanation of the Personnel Project Deployment Plan
A vertical red line is drawn in front of today's date in the diagram. Everything to the left of the line corresponds to the actual data, everything to the right of the line corresponds to the planned target data.

A green horizontal line shows the employee's planned hours per time unit of the x-axis (hours, weeks, months, years).

On the y-axis (high value) the working time (= potential) is displayed in hours (h), on the x-axis (right value) the calendar time.

Approx. 10 colours are given for the display. If more than 10 colors are needed then the colors are repeated with a slight change in brightness.

NOTE:
Caution: In many projects, the color can very quickly become indistinguishable. Please limit in such a case the displayed data by means of filter, period etc.
