Completion > Projects > Archive Projects

Completed projects can be archived (and thus possibly filtered out)


With the archiving of completed parent projects, the filter can be used to reduce the display list of projects.




Project list

Listing of all authorized mother projects with the possibility to archive or de-archive completed projects.

Listed are parent projects to which one has the right. The list can also be restricted by the filter.

Setting the archiving indicator only serves to reduce the project list in selection fields or in the project menu. It has no effect on existing data.

NOTE:
Only projects that have already been completed can be archived.



Field Descriptions

Name of Project:

The name of the project or the description of the activity (project, subproject, task) is displayed here.

Status:

This field shows how far a project has already progressed. In input mode, you can assign an attribute from a selection list to a project here, which reflects whether or how far a project is already advanced.

The status can also be used to decide whether project data should be included in a report or not.

Pln. Start:

The target start date of the project is displayed or specified here, i.e. from when the project is to be started according to planning.

Pln. End:

The target end date of the respective reference object, e.g. project, time data, trip data, etc. is displayed or specified here, i.e. by when the object should be completed according to planning.

Act. Start:

The actual start date of the respective reference object, e.g. projects, time data, trip data etc. is displayed or specified here.

Act. End:

The actual end date of the respective reference object, e.g. project, time data, travel data, etc. is displayed or specified here.

Archived:

By setting this "archiving indicator", complete mother projects can be archived, i.e. the data is retained, but these projects are no longer displayed in the selection fields or in the project menu. This makes it possible to reduce the list of projects.
A mother project can only be "archived" if it has already been completed, i.e. the status of the mother project and consequently of all subordinate projects must be set to "completed"
.
