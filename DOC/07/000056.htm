Administration > Profile/User > Profile Program Assignment

Assign individual program functions to a profile with definition of rights (display, create, change and delete)


Program functions (individual programs) can be assigned to user profiles here. If, for example, an employee profile is created, the individual program functions are assigned by mouse click. It is possible not only to assign the program function, but also to assign the individual processing modes for each program function (if available), e.g. display, create, change or delete.




Select profiles

Select one or more profiles to which program functions are to be assigned.



Field Descriptions

User Profile Groups:

In this field you can select the desired user profile group via a drop-down menu.


Select

Clicking on this button will forward the selected data.




Set program permissions

All program functions are displayed here.
Permissions for individual program functions can be assigned to the selected profiles.
Program functions displayed with blue text are program functions created by a user of the system using the TR 201 program.



Field Descriptions

Program Name:

Name of the program function

Button to save data.
