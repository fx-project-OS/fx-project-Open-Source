Interfaces > Persons > Export (XML)

Export person data as XML


Each person type created in fx-project (client, contract partner of external employees, employee + contact) can be exported comfortably with this interface. All selected persons (optionally also with address or communication data) are converted via a simple element interface into an XML file, which can be downloaded immediately and integrated in the corresponding target program.
