System requirements

System requirements for fx-project are a supported operating system, the interpreter language PHP, a web server, a database system and a network card.

Here is the exact description of the requirements:


SERVER

Operating System:Windows Server 2012 or higher, Windows 7 or higher, Linux or Unix
Webserver:IIS Server: 7 or higher, Apache: 2.4 or higher (others on request)
DB System:PostgreSQL: 8 or higher, SQL Server: 2012 or higher (others on request)
Script Interpreter:PHP 8.3 or higher incl. IonCube PHP Loader for special programs
Network protocol:TCP/IP (SSL optional)
Network card:default

CLIENT

Webbrowser:at least Internet Edge, Firefox, Opera, Google Chrome (HTML 5.0 compatible browser, JavaScript required)
