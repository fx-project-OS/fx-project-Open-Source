Reporting > Progress > Project Progress Request

Send a progress estimation request to assigned project members of a task


The project manager or project leader can use simple masks here to request his assigned resources by e-mail to provide a current progress estimate for certain tasks. This request is then also visible in the control center of the corresponding persons.




Selecting an active task

Selection of an active task for which a request for a progress estimate is to be sent.

The field Project/Task lists all active tasks to which you are assigned as project manager, project leader or deputy project leader and whose relative degree of completion is not determined with the standard method "Comparison of target/actual effort".

After selecting a task and entering an optional additional email text in the Additional Text field, clicking the Display button displays the send list in another screen.

NOTE:
Tasks for which requests must be sent according to the definition are displayed with a darker background color and an exclamation mark {!} at the end.



Field Descriptions

Tasks:

Here you can define whether tasks (and possibly to-dos) should also be displayed in the project selection list or not. If this checkbox is not set, only projects and subprojects are displayed.

Add. Text:

Input field for any text.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Send list with resource listing

Send list with resource listing

All assigned resources are listed here for the previously selected task, including a display of when a progress estimate was last specified by the person.

After a click on the button Send Question all ticked persons are requested by e-mail to submit a progress estimate. In addition, this request also appears in the control center of the corresponding persons.



Field Descriptions

Only Cust. Data:

Activate or deactivate an entry by placing a tick in the "checkbox".

Name:

In this field the family name of the person or the name of the legal entity (e.g. company name) is displayed or entered.

Statu:

Indicates the (completion) status of a ToDo item.

Day:

Full day of the week name.

Date:

Used to enter or display a date of the day. If a date entry is required, the default value (today's date) is usually already preset.


Send

By clicking on this button the entered question will be sent, i.e. saved in the internal file "Question" for later processing.
