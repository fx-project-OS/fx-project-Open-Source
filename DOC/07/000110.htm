DMS > Management > Document Manager

Upload, view, download and manage documents


Entering documents into the DMS (Document Management System) of fx-project. A file is uploaded and provided with adjustable criteria such as "category", "type", "storage location" or "description".

With the help of these criteria, documents can be easily found again. An individual assignment of rights to documents is also possible with the program function Accept.
