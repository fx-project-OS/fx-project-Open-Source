Baseboard

The footer at the bottom of the screen provides useful user information and setting options, which are described in detail below.





user information (bottom left)

#### Bild [additional info] fehlt! ###

Client number and logged in user are always visible. If you move the mouse cursor over the display, detailed user information is shown in a tooltip (e.g. role, email, etc.).


Language



The current language setting is displayed here by the respective country flag. By default, "German" is stored as the base language. In the special programs, "English" can be added as an additional option. By clicking on the respective country flag, the language setting can be changed at any time.


UID - (User Interface Design)



Clicking on the "UID icon" opens a design menu where you can choose from predefined and, if necessary, individually created design templates. This setting remains even after logout and is already preset for the next login.


Personal unit



In various program functions, it can be useful to be able to quickly adjust the person unit for calculating project effort. Depending on the client setting value (Administration>Client>Client Settings), the unit can be changed immediately with a click on the icon in the footer.

Time span format



The currently set time span format is displayed here. In various program functions it can be useful to be able to quickly adjust the time span format, i.e. clicking on the format in the footer will toggle between hour format "0:00" and decimal format "0.00".
A time span of 1 hour and 15 minutes is displayed in hour format as "1:15", and in decimal format as "1.25".


Page status



The page status indicates whether a page has just been reloaded, or whether a user action such as a click or similar has already occurred. This information can be important for test automation software to determine if a page has already been saved and rebuilt.

The following two states exist:

New, meaning no user action has been taken yet.
Edited, i.e. a user action has already been performed.


Date & time



The current date and time are displayed. If you move the mouse cursor over the date, you will get the time difference to the server where fx-project is hosted as an information tooltip. This information is important if you are in different time zones or if there has been a time change from summer to winter time or vice versa.


Response time



Specifies the duration of the program processing time for the called program function.
