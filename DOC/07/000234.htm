Completion > Projects > Destroy Projects

Completely delete projects including all assigned information from the database


Complete and final deletion of a selected project including all dependencies from the database. Before the actual deletion process, an overview is displayed showing the effects of the deletion.


NOTE:
Before shredding a project, the database should always be backed up.
The shredding of a project also deletes all associated history tables, so that a recovery of accidentally deleted data is only possible by a complete restore of a previously backed up database (see also: Database backup).




Select project

Selection of the project to be processed. Shredding completely deletes all assigned entries of the respective project, including all history data!



Field Descriptions

Project:

The Project field lists the tasks (projects or tasks) for which you have the appropriate rights and the corresponding filter criteria apply.


Select

Clicking on this button will forward the selected data.




Project statistics

Listing of the statistics data for the selected project with the option to completely remove the project including all dependencies.

NOTE:
This action cannot be undone! Please make sure to back up your database.

By clicking on this button, the above mother project including all dependencies will be completely removed from the database.
