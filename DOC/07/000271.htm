Reporting > Costs > Budget Report

Evaluation (monthly, quarterly or annual reports) of the target/actual times, as well as internal and external budgets incl. invoice list


Project reports (monthly, quarterly or annual reports)

Display of detailed project figures regarding used budgets in a specified period of time.




Selection criteria

Define criteria for limiting the project report (project, client, display type, time period, etc.)



Field Descriptions

Project:

The Project field lists the tasks (projects or tasks) for which you have the appropriate rights and the corresponding filter criteria apply.

Customer:

Here you can select the sold-to party of the task (project, subproject or task).

Display Type:

Selection field for selecting in which time display type the report is to be displayed. Monthly, quarterly or yearly.

Prj.-Category:

Define filter categories for projects or select them in a selection field.

Time Span from:

This field displays the point in time from which data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Text:

Text field

Variable text field that can be customized programmatically.

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Time auto.:

If this checkbox is ticked, the necessary period is determined automatically.

Adjust Rounding Val.:

If this checkbox is selected, an attempt is made to adjust rounded values.

Example:
A time entry of 0,25 hrs with an hourly rate of 86,50 €/hour = 21,625 € budget consumption, i.e. rounded 21,63 €. However, a back calculation from the budget consumption to the hourly rate results in 21.63 € / 0.25 hr = 86.52 €/hour.

With activated "Adjust rounding values" this hourly rate is compared with assigned hourly rates and corrected to the closest one.

Actual Effort:

The actual effort corresponds to the sum of the work actually done on tasks from time, travel or material recording. For projects/sub-projects, this value corresponds to the sum of all subordinate tasks.

Int. Used Total Bud.:

This value specifies the total internally consumed net budget (actual), that is, this is the sum of internally consumed person budget + internally consumed material budget.

Ext. Used Total Bud:

This value specifies the total externally consumed net budget (actual), that is, this is the sum of externally consumed person budget + externally consumed material budget.

Cat.:

In this field, the category is selected via a selection field.

Inv.-Margin:

Invoice margin for a project, i.e. the difference between ext. settled budget and int. consumed budget.
If no invoice writing is used, this value is always negative, if int. budget was consumed.

Inv. Position:

This field contains the unique identification number of the corresponding item in the invoice.

If this checkbox is ticked, all matching invoice positions will be listed in the report.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.

This is the monetary value, the amount of which shows the financial expenditure for the company per hour of use of a worker or a material (counterpart to VK=sales price).

Int. Used HRB:

This value indicates the internally consumed net budget (actual) for the persons who have already executed assigned tasks. The consumed budget is determined by the time recording of the employees.

Int. Used PII:

This value indicates the net (actual) budget consumed internally for the person input invoices paid for tasks already assigned. The consumed budget is determined by the entered incoming invoices.

Int. Used TCst.:

This value specifies the internally consumed net budget (actual) for trips that were posted to assigned tasks. The consumed budget is determined by the travel expense entry of the employees.

Int. Used MRB:

This value indicates the net (actual) budget consumed internally for materials that have already been used for assigned tasks. The consumed budget is determined by assigning materials to tasks or by resource planning.

Int. Used MII:

This value indicates the net (actual) budget consumed internally for the material purchase invoices paid for tasks already assigned. The consumed budget is determined by the incoming invoices entered.
