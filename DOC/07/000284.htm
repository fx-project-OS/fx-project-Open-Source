Administration > Profile/User > User Authentication

Manage 2FAs (2-Factor-Authentications) of individual users


The 2-Factor-Authentication Management is used to create, change or delete users' 2FA Device IDs.




User Selection

The user criteria for which the Device-IDs for 2FA are to be managed can be selected in this mask. The search text and the selected users are linked with "OR", i.e. if one of the two conditions applies, the user is listed.



Field Descriptions

User Name:

In this selection field, one, several or all users can be selected for which the Device-IDs are to be managed.

Search Text:

Character string to be searched for. This can be a part of the user or person name as well as a part of the Device-ID.


Select

After clicking on this button, the users with the corresponding criteria are listed.




2FA-Device-IDs

2FA-Device-IDs of Users



Field Descriptions

User Name:

This column contains the user name - the corresponding person is shown as a heading in the line above.

?:

Traffic light icon, whose tooltip displays information regarding the Device-ID for this user, i.e. whether a Device-ID is saved or whether 2FA is activated or what effects a change to a Device-ID has, etc.

2FA Device ID:

Device ID for the 2FA (2-Factor-Authentication) of a user.

Changed by:

This column shows who last made changes to this entry and when.

Delete:

Clicking on this delete icon removes the Device-ID for this user and possibly also for other users with the same Device-ID. Users with activated 2FA but without an assigned Device-ID must register their device the next time they log in.


Save

By clicking on this button, the changes are made, i.e. new Device-IDs are assigned or changed.
