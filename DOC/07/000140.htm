The structure of individual database tables in fx-project can be listed here at field level with attributes and description.




Select table

Select table
Select a table.



Field Descriptions

Table:

In mask this field is a selection field to select a certain individual table.

If no individual table is selected, text file will be created for all individual tables. Alias to ID_Tables.

Table Type:

Type of table.
1 - SystemTable,
2 - Data,
3 - Test,
4 - Individual.

Complete:

Clicking this button will list the complete content.


Display




Definition of the table fields

Definition of the fields of a table.

All fields of a table are displayed.
You can also see in which tables the respective field is still used.
If there are alias names for a field, these are shown in brackets in the respective table. The table name is stored with a link and can be displayed in a new window.



Field Descriptions

PKey:

Primary key in corresponding table.

Pos.:

Position of this value in the listing, i.e. where the entry should be placed.

Field Name:

Specifies the name of the field.

When creating a field, make sure that the name is unique.

FieldType:

In the case of "DATE" and "TIME", this field determines how the mask generator formats the corresponding field. An alias field must be defined for the field to be formatted using SQL.

Length:

Length of the field. For most fields, certain restrictions apply here (e.g. text maximum 256 characters long).

If one of these restrictions is violated, the program automatically takes over the correction.

NNull:

This field must be output,

Index:

blank

Clst.:

Indicates whether the field may contain duplicate values.

Description:

Explanatory descriptions, details, comments or similar.
