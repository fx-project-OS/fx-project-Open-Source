Administration > Skills > Skills

Freely definable skills, i.e. learned or acquired knowledge


Management (creation, modification or deletion) of freely definable skills. These skills can then be assigned to individual employees and evaluated in the corresponding program function (= skill management).




Select group

Select group

Here the user can select a group and display the capabilities for it.



Field Descriptions

The individual ability groups of the employees can be selected.:

In this field, the individual employee skill groups can be selected, i.e. the employee areas they cover. The selected group ID is stored internally.


Accept

Clicking this button saves the entered data and continues the program execution.




Administration

Skills

The selected skills are displayed, changed or deleted here.



Field Descriptions

Only Cust. Data:

Activate or deactivate an entry by placing a tick in the "checkbox".

Skill Name:

This field contains the names of the skills of the individual persons, e.g. "English", "Russian" or similar as skills for the group "foreign languages".


Save

By clicking on this button, the changes are made. The text of the button depends on the current action mode.
