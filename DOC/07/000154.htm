Approval of the recorded material consumption by employees or superiors


Release of material consumption entries in the corresponding control instance. This means that an entry should run through the release procedure before a possible settlement. This ensures that correct data is available.

The control instances are arranged in the following sequence:
(1) MA=employee - (2) PL=project manager - (3) VW/GL=administration/management.




Selection criteria

Search criteria can be used to restrict the number of entries to be released (materials or optionally also time data or travel expenses). Search criteria can be e.g. the period (normally the last month), a certain project, a certain person or a certain client.

Furthermore, a checkbox can be used to specify whether entries that have already been released should be displayed or hidden.



Field Descriptions

period of time:

Specify the time period (start and end date) in which found entries are to be displayed. Normally this period is preset with the last month.

CAUTION! If the period is long, this can lead to considerable waiting times.

Text:

Text field

Variable text field that can be customized programmatically.

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Release Hide entries:

If this checkbox is set, already released entries in the selected period will be hidden, i.e. not listed.

Project:

If a task or a project is selected here, only entries are listed which were booked directly to this task or indirectly to a subordinate task of this project.

In the selection field, only projects can be selected for which one has the corresponding rights and for which the corresponding filter criteria apply.

Max. Project Chars:

Definition of the maximum number of text characters to be displayed for the project designation, i.e. if a text is too long (in particular for print formatting), the text is wrapped or truncated from the number of characters entered, depending on the "Line break" field set. If a value is entered, the project is also displayed without the project numbering.

An entry of '0' means: No text wrapping and project numbering (= default).

Person:

If a person is selected here only entries are listed, which were booked directly for this person.

In the selection field, only persons can be selected for which one has the appropriate rights and for which the corresponding filter criteria apply.

Max. Text Chars:

Defines the maximum number of text characters to be displayed, i.e. if a text is too long (especially for print formatting), the text will be wrapped or truncated after the entered number of characters, depending on the "Line break" field set.

An entry of '0' means: no text wrapping. For print formatting, e.g., for the "Proof of service", the preset value of 50 (= default) is recommended.

Customer:

If a client is selected here only entries are listed, which were booked directly for tasks, which are assigned to this client.

In the selection field, only clients can be selected for which one has the corresponding rights and for which the corresponding filter criteria apply.

Line Break:

Specifies whether texts are to be wrapped or truncated after a certain number of characters. When text is wrapped, the text remaining after a certain number of characters is continued in the next line and possibly wrapped again, and so on.

List time data.:

Specify whether or not time entries should be listed. (In the time data release, this value is always set.)

List travel expenses.:

Specify whether or not travel expenses should be listed. (In the travel expenses release, this value is always set.)

List material consumptions.:

Specify whether material consumptions should be listed or not. (In the material release this value is always set.)

Display annotations:

Specify whether or not any internal notes entered should be listed in addition to the descriptions.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.




Approval

List of recorded times, travel expenses or material consumption depending on defined selection criteria. The individual entries can then be released. Released entries can no longer be changed by a lower instance. If authorized, the release can also be reversed.



Field Descriptions

(enable):

Entries marked here are released after clicking the [Save] button.

Date:

Current date of the listed row.

Time:

Time or time period of the time entry or time material of the listed row.

Project:

Assigned task of the entry of the listed row.

Type:

Type of entry of the listed row, ie.
Times:
Time entry

Travel:
Domestic travel
Foreign travel
Receipts without travel
- Daily details
- Receipts

Materials:
Immediate materials
-Time or consumable materials

Other:
Other

Description:

Explanatory descriptions or details of the entry of the listed line.

Internal Annotation:

Additional (mostly internal) annotations of the entry of the listed line.

(This column is only shown if the Show annotations field was checked in the selection mask.)

Amount:

Number of material consumption of the entry of the listed row.

Unit:

Unit of the material consumption of the entry of the listed line.

TR ni:

Indicates whether the times of this entry of the listed row are billable (= ) or not (= ).

If authorized, this property can be modified by checking or unchecking the appropriate checkbox.

TC ni:

Indicates whether the travel expenses of this entry of the listed row are allocable (= ) or not (= ).

If authorized, this property can be changed by checking or unchecking the corresponding checkbox.

MU ni:

Indicates whether the material consumption of this entry of the listed line is allocable (= ) or not (= ).

If authorized, this property can be changed by checking or unchecking the corresponding checkbox.

Status:

Release status of the posting record of the listed row, thereby means:
MAAs last instance released by employee
PLAs last instance released by. Project manager released
VW/GLAs last instance released by administration or management


Save

By clicking on this button the releases and possible chargeable statuses are changed.
