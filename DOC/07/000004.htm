Program start and program end

fx-project is called up via the link address in the browser that was provided to you by the administrator.

The user login mask is displayed on the start page. Enter the client no. assigned to you here, If you have forgotten the password, contact your administrator to reset the password.

You can access the fx-project application (= login) by clicking on the [Login] button or by pressing the [Return] key .




To optionally exit the program (= logout/logout), click the logout button in the top right navigation toolbar.

If you do not log out, fx-project remembers your login data for today, so you can continue working where you left off without logging in.
