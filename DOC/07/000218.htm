Creation > Employees > Weekly and Overtime Hours

Setting of an internal employee's overtime (+/-) at a specific time


In order to record any overtime of an employee, the start time (date) and the start number of hours of the calculation are defined here. From this point on, the defined target working time is compared with the actual times of the employees from the time recording. The result corresponds to the overtime of the employee.




Person selection

Define selection criteria for filtering the persons to be listed for whom the weekly and overtime hours are to be displayed and/or defined.



Field Descriptions

Search Text:

This field is used to enter the string to search for.

Time Span from:

This field displays the point in time from which data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Active Employee:

This checkbox can be used to select whether all employees who match the selection criteria should be displayed or only those employees who have a displayable active record in this period.

Checkbox ticked = Only employees who have a displayable active record.

Department:

Indicates the department in which the employee works.

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Display History:

If this checkbox is ticked, an additional history is displayed in addition to the normal data.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.

Shows which employees are assigned to a project as workers.

In the case of (sub-)projects, all employees who are assigned to the respective subtasks are displayed here.

EDat.:

The entry date of a person, i.e. since when the employee has been employed by the company, must be entered in this field.

Hrs/W:

This field displays the number of working hours to be worked by an employee (per week) and on which days the employee is available as a resource.

Month:

In this field, the calendar month is output via a selection menu.

Year:

The year is specified in this field.
