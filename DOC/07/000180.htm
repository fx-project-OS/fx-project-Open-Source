Workspace

The fx-project workspace is located in the middle of the screen between the top menu bar and the bottom footer bar.


The work area naturally takes up the most space and usually consists of five parts:

Breadcrumb (top left)



Navigation path to the current program function, for example, to quickly find a higher-level path, or to be able to call up the current program function again.

Breadcrumb (top left)



Navigation path to the current program function.For example, quickly a parent path, or to be able to call the current program function again.


Action tab (top right)



Listing of possible and allowed actions for the program function. Here you can choose between [View], [New creation], [Modify] or [Delete].


Program name and mask buttons (frame header above)



After entering the data in the masks, the selected action is executed after clicking on the corresponding "mask button" at the top right of the frame header, for example "Save".

The whole workspace can be maximized or
subsequently normalized again by clicking on the icon.


Mask(s) (Main Area)


The "Mask Area" displays the masks for the called program functions. A program function can consist of one or more submasks.


Feedback slider (right)


The "Feedback slider" displays success and/or warning or error messages, which occurred after the performed action triggered by the click on a "mask button".
