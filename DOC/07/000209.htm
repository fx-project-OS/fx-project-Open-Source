Project overview of all information on a project, subproject or task


Detailed graphical overview of a selected project including further links. The structure is divided into the following three main areas:

  • Project:
    General information, project progress, documents, target-performance comparison and degrees of completion.
  • Persons:
    Main project managers, additional project managers as well as executing persons (= resources).
  • Costs:
    Budget type, internal and external target budget, margin as well as invoice data.




Project overview-selection

Please select a project or task for which you want to see a short overview with navigation links. Then click on the [Display] button.



Field Descriptions

Project:

The Project field lists the tasks (projects or tasks) for which you have the appropriate rights and the corresponding filter criteria apply.


Display

By clicking on this button, the list is generated and displayed. Depending on the defined selection criteria, the list is filtered accordingly.
