Personnel main sheet of an internal employee


A clear arrangement of all relevant data such as employee data, contract data, bank data, address data or communication data for one person.

The amount of information can be easily increased or decreased by ticking the corresponding checkboxes, and the period of the special data to be displayed, such as time data or absence data, can be freely defined.




Display criteria for the person master data sheet

In the mask Display criteria for the person master data sheet the selection is made which information is to be listed to a person. Prerequisite is the restriction to a certain person, as well as to a certain period. The employee data are always displayed, no matter which selection criteria are set.

The period to be displayed must be specified using the two input fields Time Span from and Time Span to. If no entry is made, the current calendar month is preset.

By selecting the fields Time Data, Holidays, Sickness or Training, the corresponding data for the selected person is determined and displayed in total. Here, [Total] means display of the subtotal per year and [Detailed] per calendar month.

By ticking the checkbox fields Contract Datas, Bank Datas, Addresses, Communication Data, Skills or Personnel Hierarchy the corresponding data for the selected person will be determined and displayed in the same overview block.

The desired data are determined after a click on the button Display and displayed in the mask Personnel master sheet, where they can be processed further via the [File Manager].



Field Descriptions

Person:

This selection field is used to select the person whose data is to be edited or displayed depending on the action.

Time Span from:

This field displays the point in time from which data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Time Span to:

This field displays the date up to which the data for a project should be listed.

In input mode, a date can be entered here according to the user settings.

Time Data:

In the selection field Time Data it can be specified whether and how the working hours of the person in the selected period should be displayed. Total] means per calendar year, [Detailed] means per calendar month.

Holidays:

In the selection field Holidays it can be specified whether and how the vacation days taken by the person in the selected period should be displayed. Total] means per calendar year, [Detailed] means per calendar month.

Sickness:

In the selection field Sickness it can be specified whether and how the sick days of the person in which he/she was absent should be displayed in the selected period. Total] means per calendar year, [Detailed] means per calendar month.

Training:

In the selection field Training it can be specified whether and how the days of the person in which he/she was on training should be displayed in the selected period. Total] means per calendar year, [Detailed] means per calendar month.

1509:

Selection of all created project category groups

Contract Datas:

By checking the Contract Datas field, you can determine whether the corresponding data should be listed in the block with the same name or not.

Bank Datas:

By checking the Bank Datas field, you can determine whether the corresponding data should be listed in the block with the same name or not.

Addresses:

By checking the Addresses field, you can determine whether the corresponding data should be listed in the block with the same name or not.

Communication Data:

By checking the Communication Data field, you can determine whether the corresponding data should be listed in the block with the same name or not.

Skills:

By checking the Skills field, you can determine whether the corresponding data should be listed in the block with the same name or not.

Personnel Hierarchy:

By checking the Personnel Hierarchy field, you can determine whether or not the corresponding data should be listed in the block of the same name.
Personnel Hierarchy here means both superior and subordinate.

TR n. invoice.:

Here you can specify whether the time recording should be invoiced or not.
