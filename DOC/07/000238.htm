Interfaces > Persons > Import (XML)

Import person data as XML


Each person type created in fx-project (client, contract partner of external employees, employee + contact) can be imported comfortably with this interface. The persons to be imported (optionally also with address or communication data) must be available in a fx-project-compliant XML file. This makes it very easy, for example, to transfer persons from one client in fx-project to another.
