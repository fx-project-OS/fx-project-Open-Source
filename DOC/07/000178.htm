Project menu (project slider)

The project menu, or project slider, is located on the left side of the screen and is used for quick project entry at any level.

The project slider is the bottom of the two slider elements on the left.

Approaching with the mouse cursor, the subtle "tab" becomes larger and labeled. By clicking on this "tab", the project menu can be expanded and collapsed.

Opened, the user gets an overview of the project structure of all authorized projects. The number of projects displayed can be controlled by the "Filter".



Within the project workflow menu, the user has the following control options:

Expand all projects.
Collapse all projects.
Refresh Project Menu
Sort by project name.
Sort by project number


Individual project levels can be expanded or collapsed by clicking on the respective nodes.

By clicking on the project number of the desired project, all project sublevels of this project can be expanded or collapsed at once.

The following options may be available for the respective action:

Call program function in the mode: Display.
Call program function in mode: New creation
Call program function in mode: Change
Call program function in mode: Delete
