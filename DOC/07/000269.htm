Creation or modification of fixed fees per km and non-monetary renumerations


Settings for travel expense per diems, such as mileage allowance or meals.

The automatic calculations of the travel expense program are based on the travel expense flat rates stored here.

In this program function you can store the currently valid statutory mileage allowances for the various vehicle types and passengers, as well as per diems and reductions for meals for domestic and international trips.

This program function is a standard administrator program and can only be customized by individuals with appropriate administrator privileges.

NOTE:
Make sure to always store the currently valid statutory per diems, otherwise trips can no longer be accounted for correctly for tax purposes! These can be found in the official country list of the Federal Ministry of Finance.




Settings

Here, various settings can be changed according to personal preferences, e.g. the "Date format", the "Decimal separator", the "Number of decimal places", the "Number of list entries", the "Sort order of projects", etc.

Grayed settings have not yet been adjusted, i.e. these contain the specified original default values of your client.

User settings that have already been adjusted can be reset to the original default value by checking the "Delete" checkbox in front of the corresponding setting value.

A click on the "question mark" at the end of the line opens a separate help window for the corresponding setting value with an exact description of the change options.
