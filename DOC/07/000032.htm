Execution > Appointments > Search

Search and list certain appointments


Convenient search of appointments according to criteria that can be combined with each other. These criteria are "Person", "Subject", "Period" and "Appointment type".
All appointments found are listed clearly below each other according to person and period. If the authorization is available, the displayed appointment is linked with a change option.




Search criteria

Search criteria

Search for specific appointments using search criteria, e.g.
by a part of the subject of the appointment or
by a specific appointment type.

Of course, you can also combine different search criteria.
After clicking "Search", the system searches for the appointments that meet the criteria.



Field Descriptions

Person:

This selection field is used to select the person whose data is to be edited or displayed depending on the action.

Reference:

Specifies the subject of an appointment.

Search fr.:

In this search field you can define the validity start of the time frame of the search.

Search to:

In this search field you can set the validity end of the time frame of the search.

App. Type:

The field "Appointment type" provides different appointment types for selection. While project managers can view all business appointments of an employee, you are not allowed to control private appointments.


Search

Clicking on this button starts the search.




Search results

Search results

All hits for a search are displayed in a list.
By clicking on the button OK (to the right of each hit), you will be automatically forwarded to this appointment.



Field Descriptions

Start of App.:

This field displays the start time for an appointment.

End of App.:

This field displays the end time for an appointment.
