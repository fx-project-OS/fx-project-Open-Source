Erfassung > Mandant > Grunddaten

Festlegen der Hauptinformationen des Mandanten-System


Verwaltung (Neuanlage, Änderung oder Löschung) der Grunddaten von Mandaten wie Name, Personenbild, Adress- oder Kommunikationsdaten. Das Löschen eines Mandanten vernichtet alle vorhanden Daten dieses Mandanten. Der Mandant Nr. 1 ist unverzichtbar um mit fx-project arbeiten zu können und kann demzufolge nicht gelöscht werden.




Auswahl

In dieser Teilmaske wird die zu bearbeitende Person/Firmenname ausgewählt. Handelt es sich um eine Programmfunktion für Mitarbeiterdaten (ext./int.) so ist zusätzlich das Auswahlfeld inaktiv zur Auswahl aktiviert. Ist das Feld inaktiv aktiviert, werden auch Personengruppen die nicht mehr aktiv sind angezeigt.



Feldbeschreibungen

Mandant:

Auswahl des Mandanten, welcher bearbeitet oder angezeigt werden soll.

Kommunik. sortieren:

Einstellung, wie die anzuzeigenden Kommunikationsdaten sortiert werden sollen.


Auswählen

Nach einem Klick auf diesen Button werden die entsprechenden Daten des ausgewählten Mandanten geladen und zur Bearbeitung oder Betrachtung angezeigt.




Zusammenfassung der Informationen

Personenübersicht

In der Maske Zusammenfassung der Informationen erhält man einen gesamten Überblick über die über- und untergeorneten Personen.



Feldbeschreibungen

Übersicht:

Übersicht gibt eine kurze Übersicht über die gewählte Person.




Definition

Hauptdaten eines Mandanten

Definition speichert die wichtigsten Daten zu einem Mandanten und definiert ihn somit.

Im Neuanlage-Modus muss außerdem ein Administrator angelegt und ein Mandant von dem die Administration>Mandant>Nachschlagewerte übernommen werden sollen, gewählt werden.

Nach dem erfolgreichen Anlegen kann sich der Administrator sofort im neuen Mandanten einloggen. Hierzu entnimmt er der Bestätigungsmeldung die neue Mandanten ID.



Feldbeschreibungen

Mandant-Nr:

Interner Schlüssel zur Identifizierung eines Mandanten. Ein Mandant ist ein rechtlich eigenständiges Unternehmen.

Organisationseinheiten innerhalb eines Unternehmens gelten nicht als Mandant.

Name:

In dieses Feld wird der Name des Mandanten eingetragen.

Bild:

Zugeordnetes Bild im Format PNG, JPG, GIF oder BMP.

Name Zusatz:

Administration>Fähigkeiten>Fähigkeitsgruppen ist eine Erweiterung des Feldes Kontrolle>Allgemein>Kontrollzentrum. Hier können zusätzliche Informationen gespeichert werden, die evtl. im oberen Feld keinen Platz hatten.

Identifikation:

Identifikations-Nr speichert Initialen, Kundennummern oder jegliche Art anderer Nummern zu einer Person.

Tabelle für frei veränderliche Werte:

Tabelle für frei veränderliche Werte. Hier kann der Benutzer Tabelleninformationen individuell erweitern, z.B. Adresstyp, Anrede, Titel von Personen etc.

Gültig von:

Datum, ab wann ein Datensatz gültig ist.

Gültig bis:

Datum, bis wann ein Datensatz gültig ist.

inaktiv:

Durch einen Haken in Abteilung werden Personen gekennzeichnet, die ausgeblendet werden sollen (ausgetretene Mitarbeiter, alte Kunden, ...).
In Auswahl angehakt, bewirkt das Feld auch, dass die inaktiven Personen mit dem Zusatz "inaktiv" angezeigt werden.

Inaktivsetzen eines Mandanten bewirkt außerdem, dass dieser komplett gesperrt wird und niemand sich mehr einloggen kann.

Benutzername:

In diesem Feld wird der Benutzername in Textform angezeigt bzw. in Textform eingegeben (maximal 20 Zeichen).

Dieser kann nach der Namenskonvention des Mandanten aufgebaut werden und wird normalerweise vom Administrator verwaltet.

Vorschläge für die Namenskonvention: Vorname, Zuname, Initialen oder ein vom Benutzer selbst gewählter Name.

Passwort:

In dieses Feld muss ein persönliches Passwort eingegeben werden. Das Passwort sollte zur eigenen Sicherheit mindestens sechs Zeichen lang sein, kein "$"-Sonderzeichen etc. enthalten, sowie aus keinen leicht zu erratenden Buchstaben- bzw. Ziffernfolgen bestehen wie "123456", "654321", "qwertz" etc.

Das Passwort wird intern verschlüsselt gespeichert, so dass bei einem eventuellen "Vergessen" ein neues Passwort beantragt werden muss.

E-Mail-Adresse:

In diesem Feld wird die E-Mail-Adresse angezeigt bzw. eingegeben

Passwort wdh.:

Bitte geben Sie das neue Passwort nochmal ein, damit Tippfehler ausgeschlossen werden können.


Multi-Funktion

Durch Klick auf diesen Button werden die Änderungen durchgeführt.

Zusatz löschen

Zusatzdaten (= Adress- und Kommunikationsdaten) der Person/Firma löschen

Durch einen Klick auf den Button Zusatzdaten löschen werden alle ausgewählten Zusatzdaten, d.h. Adress- und Kommunikationsdaten der Person/Firma gelöscht.

Firma löschen

Gesamte Person oder Firma löschen, inkl. aller Zusatzdaten.

Durch einen Klick auf den Button Person löschen wird die gesamte ausgewählte Person bzw. Firma inkl. deren Zusatzdaten wie Bilder, Adress- und Kommunikationsdaten gelöscht.
Eine Person bzw. Firma kann nicht gelöscht werden, wenn noch untergeordnete Personen hierzu bestehen.




Adressen

Adressenverwaltung

Die Teilmaske Adressen speichert Adressen zur gewählten Person/Firma. Die Anzahl der leeren Einträge kann über die Programmfunktion Administration>Benutzer>Benutzereinstellungen bestimmt werden. Die verschiedenen Arten von Adressen werden in Administration>Mandant>Nachschlagewerte mit Adresstyp gepflegt.



Feldbeschreibungen

Nur Kundendaten:

Durch ein Häkchen in Checkbox wird gekennzeichnet, dass dieser Eintrag gelöscht werden soll.

Adresstyp:

In Adresstyp wählt man den Adresstyp der Anschrift aus.

Staat:

Staat beinhaltet den Staat der Anschrift.

Straße:

Straße beinhaltet Straße und Hausnummer der Anschrift.

Land:

Länderkennzeichen beinhaltet das Länderkennzeichen der Anschrift.

PLZ:

Erfassung>Ext. Mitarbeiter>Rollen beinhaltet die Postleitzahl der Anschrift.

Ort:

Ort beinhaltet den Ort der Anschrift.

Kommunikation

Kommunikationsverwaltung

In Maske Kommunikation können alle relevanten Informationen, wie eine Person erreicht werden kann (außer der Adresse), gespeichert werden. Neue Typen werden in Administration>Mandant>Nachschlagewerte mit Kommunikationstyp gepflegt.



Feldbeschreibungen

Nur Kundendaten:

Durch ein Häkchen in Checkbox wird gekennzeichnet, dass dieser Eintrag gelöscht werden soll.

KommunikatTyp:

In Kommunikationstyp wird der Kommunikationstyp gewählt. "Standard"-Typen werden in Zusammenfassung der Informationen angezeigt.

Nummer:

Nummer beinhaltet die Kommunikationsnummer (Telefonnummer, E-Mail-Adresse, ...).

Anmerkungen:

Anmerkungen enthält zusätzliche Informationen zur Nummer (Gültigkeit, Erreichbarkeitszeiten, ...).
