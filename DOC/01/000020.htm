Abschluss > Projekte > Projekte archivieren

Abgeschlossene Projekte können archiviert (und damit evtl. rausgefiltert) werden


Mit der Archivierung von abgeschlossenen Mutterprojekten kann über den Filter die Anzeigeliste von Projekten verkleinert werden.




Projektliste

Auflistung aller berechtigten Mutterprojekte mit der Möglichkeit abgeschlossene Projekte zu archivieren oder zu dearchivieren.

Aufgelistet werden Mutterprojekte zu welchen man das Recht besitzt. Eingeschränkt werden kann die Liste ferner durch den Filter.

Das Setzen des Archivierungskennzeichens dient lediglich zum Verkleinern der Projektliste in Auswahlfeldern oder im Projektmenü. Es hat keine Auswirkung auf vorhandene Daten.

HINWEIS:
Nur bereits abgeschlossene Projekte können archiviert werden.



Feldbeschreibungen

Name des Projektes:

Hier wird der Name des Projekts bzw. die Bezeichnung des Vorgangs (Projekt, Teilprojekt, Aufgabe) angezeigt.

Status:

In diesem Feld wird angezeigt, wie weit ein Projekt bereits fortgeschritten ist. Im Eingabemodus kann man hier einem Projekt ein Attribut aus einer Auswahlliste zuordnen, das wiedergibt, ob bzw. wie weit ein Projekt bereits fortgeschritten ist.

Auch kann über den Status entschieden werden, ob Projektdaten in einen Report einfließen sollen oder nicht.

Gep. Start:

Hier wird das Soll-Startdatum des Projektes angezeigt bzw. angegeben, d.h. ab wann laut Planung mit dem Projekt zu beginnen ist.

Gep. Ende:

Hier wird das Soll-Enddatum des jeweiligen Bezugsobjekts, z.B. Projekt, Zeitdaten, Reisedaten etc. angezeigt bzw. angegeben, d.h. bis wann laut Planung das Objekt abgeschlossen sein soll.

Tat. Start:

Hier wird das Ist-Startdatum des jeweiligen Bezugsobjekts, z.B. Projekte, Zeitdaten, Reisedaten etc. angezeigt bzw. angegeben.

Tat. Ende:

Hier wird das Ist-Enddatum des jeweiligen Bezugsobjekts, z.B. Projekt, Zeitdaten, Reisedaten etc. angezeigt bzw. angegeben.

Archiviert:

Durch das Setzen dieses "Archivierungskennzeichens" können komplette Mutterprojekte archiviert werden, d.h. die Daten bleiben erhalten, jedoch werden diese Projekte nicht mehr in den Auswahlfeldern oder im Projektmenü angezeigt. Dadurch ist es möglich die Liste der Projekte zu verkleinern.
Ein Mutterprojekt kann nur "archiviert" werden, wenn es bereits abgeschlossen wurde, d.h. der Status des Mutterprojektes und demzufolge aller untergeordneten Projekte muss auf "abgeschlossen" stehen.
