Administration > Profile/Benutzer > Benutzerkonten

Aktivieren, Deaktivieren oder neues Passwort generieren für Benutzer


Benutzeraccounts (Benutzerzugänge) können bei Bedarf deaktiviert werden, d.h. die zugehörigen Benutzerprofile mit den zugeordneten Lizenzpunkten werden dem Punktekonto wieder gutgeschrieben. Dasselbe gilt für die Aktivierung, jedoch auf umgekehrte Weise. Außerdem können gesperrte Benutzer, deren Zugang nach mehrmaliger fehlerhafter Passworteingabe gesperrt wurde, wieder aktiviert werden. Zusätzlich kann auf Wunsch eines Benutzers ein neues Passwort generiert werden, welches dem jeweiligen Benutzer per E-Mail zugesandt wird.




Statussteuerung

Auswahl der Aktion

Von der Statussteuerung aus gelangt man durch die Button Passwort generieren, Sperre auheben, Benutzer aktivieren und Benutzer deaktiv. auf Benutzerstatus.



Feldbeschreibungen


Anzeigen

Passwort gen.

In Passwort generieren wird ein neues Passwort generiert und dem Benutzer per E-Mail zugeschickt.

Sperre aufh.

In Sperre auheben werden Benutzer entsperrt, die wiederholt ihr Passwort falsch eingegeben haben.

Benutzer akt.

Benutzer aktivieren

In Benutzer aktivieren werden inaktive Benutzer wieder aktiviert.

Benutzer deakt.

In Benutzer deaktiv. werden aktive Benutzer auf inaktiv gesetzt.

Datum:

Status:

In diesem Feld wird der aktuelle Status eines Benutzers gespeichert:
0 - aktiv (ohne SYS-Passwort) = ohne automatisch vom System generiertes Passwort nach Sperre etc.
1 - aktiv (mit SYS-Passwort) = mit systemgenerierten Passwort über [Passwort gen.]
2 - inaktiv
3 - gesperrt
