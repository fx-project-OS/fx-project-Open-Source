Erfassen oder Bearbeiten von Sofort-, Verbrauchs- und/oder Zeitmaterialien


Erstellung und Verwaltung von Sofort-, Verbrauchs- und/oder Zeitmaterialdefinitionen.

Sofortmaterialien sind Materialien, welche einmalig zu Aufgaben zugeordnet werden, wodurch die entsprechenden Kosten und/oder Erträge direkt ins Projektbudget übernommen werden, z.B. Computer, Drucker, etc.

Verbrauchsmaterialien sind Materialien, welche einheitsweise (Stück, Meter, etc.) von einer Person taggenau verbraucht werden können, z.B. Leitungen, Steckdosen, etc.

Zeitmaterialien sind Materialien, welche analog zur Zeiterfassung stundenweise von einer Person taggenau verbraucht werden können, z.B. Firmenwagen, Mieträume, etc.




Materialauswahl

Über Suchkriterien kann die Anzahl der Materialliste, welche angezeigt oder verändert werden soll, eingeschränkt werden. Als Suchkriterien können z.B. Artikelnummern, Materialnamen, zugeordnete Projekte, Materialtypen, Verwendungsarten, Lagerorte und/oder Start- bzw. Enddatum angegeben werden.

Materialdefinition

Sofortmaterialien sind Materialien, welche einmalig zu Aufgaben zugeordnet werden können, z.B. Computer, Drucker, etc. Wird ein Material zu einer Aufgabe zugeordnet werden die entsprechenden Kosten und/oder Erträge direkt ins Projektbudget übernommen. Ob ein Sofortmaterial abrechenbar ist oder nicht kann durch einen Eintrag in das Datumsfeld "Abrechenbar ab" festgelegt werden.

Verbrauchsmaterialien sind Materialien, welche einheitsweise (Stück, Meter, etc.) von einer Person taggenau verbraucht werden können, z.B. Leitungen, Steckdosen, etc. Über die Datumsfelder "Startdatum" und "Enddatum" kann geregelt werden in welchem Zeitraum dieses Material verbraucht werden kann. Ein Verbrauchsmaterial kann generell von allen Person verwendet und zu berechtigten Aufgaben hinzugebucht werden. Verbrauchsmaterialien können optional auch als Lagerartikel definiert werden, sodass deren Bestände in der Lagerverwaltung kontrolliert werden können.

Zeitmaterialien sind Materialien, welche analog zur Zeiterfassung stundenweise von einer Person taggenau verbraucht werden können, z.B. Firmenwagen, Mieträume, etc. Über die Datumsfelder "Startdatum" und "Enddatum" kann geregelt werden in welchem Zeitraum dieses Material benutzt werden kann. Ein Zeitmaterial kann generell von allen Person verwendet und zu berechtigten Aufgaben hinzugebucht werden. Für Zeitmaterialien können wie bei Personen (HR) auch festgelegt werden, an welchen Wochentagen und für wie viele Stunden/Tag diese zur Verfügung stehen.

Materialstatus:

Budgetstatus des Materials:

Angelegt bedeutet, dass das Material erfasst, aber noch nicht geordert, eingetroffen oder bezahlt worden ist. Die Werte haben noch keine Auswirkung auf das Materialbudget der zugeordneten Aufgabe. Damit können z.B. Materialien bereits erfasst werden, auch wenn das Materialbudget der Aufgabe nicht ausreicht oder noch nicht angepasst worden ist. Bei Verbrauchs- oder Zeitmaterialien ist dieser Materialstatus immer eingestellt, da sich hier der Materialbudgetverbrauch anhand der täglichen Verbrauchserfassung errechnet.

Bestellt bedeutet, dass das Sofortmaterial bereits geordert wurde, aber noch nicht eingetroffen oder bezahlt worden ist. Der Int-Wert (Menge x EK) wirkt sich auf das intern verplante Materialbudget der zugeordneten Aufgabe aus.

Bezahlt bedeutet, dass das Sofortmaterial geordert, eingetroffen und bezahlt worden ist. Nur ein bezahltes Material kann in einer ausgehenden Rechnung abgerechnet werden. Der Int-Wert (Menge x EK) wirkt sich auf das intern und der Ext-Wert (Menge x VK) auf das extern verbrauchte Materialbudget der zugeordneten Aufgabe aus.

Das Rechnungsdatum eines Sofortmaterials, oder das Gültigkeitsenddatum eines Verbrauchs- oder Zeitmaterials.

Kostenträger

Kostenträger:

Ein Kostenträger ist ein Objekt, dem im betriebswirtschaftlichen Rechnungswesen Kosten zugerechnet werden. Üblicherweise versteht man darunter eine Ware oder Dienstleistung. Grundsätzlich können aber im Rahmen der betrieblichen Kostenrechnung auch andere Objekte als Kostenträger gewählt werden, wie etwa ein Prozess (Prozesskostenrechnung), ein Projekt (Projektkostenrechnung) oder ein Kunde (Kundenprofitabilitätsrechnung).

Kostenstelle

Kostenstelle:

Die Kostenstelle ist ein räumlich oder organisatorisch abzugrenzender Ort einer Organisation, an dem Kosten anfallen. Kostenstellen entsprechen zum Beispiel also einzelnen Maschinen oder sonstiger abgegrenzter Materialien.

Leistungsart:

Die Leistungsart in der Kostenrechnung ist eine Bezugsgröße, die die qualitative technische Beschreibung der Produktionstätigkeit angibt. Hierbei kann klassifiziert werden zwischen produktiven (=wertschöpfenden) und nicht-produktiven Arten.
