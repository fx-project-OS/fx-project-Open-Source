Erfassung > Projekte > Auftragsanfragen

Erfassen von geplanten Anforderungen, die nach erfolgter Freigabe zu einem Hauptprojekt werden


Verwalten (Neuanlage, Ändern oder Löschen) von Aufträgen, d.h. von geplanten Anforderungen. Erfasst werden kann dabei der Titel, ein Ansprechpartner, Fertigstellungstermin, Auftragsursache und -priorität sowie Auftragsziel und Anmerkung als reine Textwerte.

Aus jedem Auftrag kann mit nur einem einzigen Mausklick ein neues Hauptprojekt erzeugt werden.




Auswahlkriterien festlegen

Auftrag direkt auswählen oder Liste durch Kriterien einschränken.



Feldbeschreibungen

Titel des Auftrages:

Im Auftragstitel steht der Name bzw. die Bezeichnung des geplanten Vorgangs (Projekt). Bei der Aktivierung eines Auftrages wird aus diesem Titel ein neues Mutterprojekt erstellt.

Auftraggeber eines Projektes.:

Hier wird der Auftraggeber des Vorgangs (Projekt, Teilprojekt bzw. Aufgabe) gewählt.

Filterkategorien für Projekte:

Filterkategorien für Projekte festlegen oder in einem Auswahlfeld selektieren.

Jahr:

In diesem Feld wird die Jahreszahl angegeben.


Anzeigen

Durch Klick auf diesen Button wird Liste erzeugt und angezeigt. Je nach festgelegten Auswahlkriterien wird die Liste hierbei entsprechend gefiltert.




Auftrags-Liste

Anzeige aller Aufträge, welche den eingestellten Kriterien entsprechen.

Durch einen Klick auf das [AKTIVIEREN]-Icon kann ein Auftrag als Mutterprojekt angelegt werden. Gleichzeitig werden hierbei alle Projektmanager per E-Mail über das neue Projekt benachrichtigt. Durch einen Klick auf das [AKTIVIEREN]-Icon können nochmals die Informationen zum Auftrag eingesehen werden.



Feldbeschreibungen

Auftragstitel:

Im Auftragstitel steht der Name bzw. die Bezeichnung des geplanten Vorgangs (Projekt). Bei der Aktivierung eines Auftrages wird aus diesem Titel ein neues Mutterprojekt erstellt.

Proj. Nr:

Dies ist die Verzeichnisnummer für ein Projekt.

Die Verzeichnisnummer kann bei Mutterprojekten nicht geändert werden.

Die Verzeichnisnummer wird beim Löschen oder Verschieben automatisch angepasst.

Durch manuelles Ändern der Verzeichnisnummer können Sie Projekte innerhalb einer Hierarchie verschieben.

Anspr. Auftrag.:

Dieses Feld zeigt den Ansprechpartner bzw. den Kunden. Durch Selektion eines Ansprechpartners wird in einigen Fällen (z.B. Projektdefinition) automatisch auch ein Kunde gewählt, da jeder Ansprechpartner genau einem Kunden zugeordnet ist.

Angelegt wird ein Ansprechpartner über die Eingabemaske im Menüpunkt "Stammdaten".

Fertigstellung:

Spätestes geplantes Datum, bis zu welchem Termin der Auftrag bzw. der zugehörige Projekt abgeschlossen sein soll.

Link:

Dieses Feld ist optional. Ist ein Link eingetragen, so wird die automatische Benutzerrecht-Abfrage übergangen und das Feld Trans_ID gelöscht.




Datenblatt

Komfortables Verwalten und Erstellen von Aufträgen.

Auf Basis des gewählten Auftrages wird automatisch ein neues Mutterprojekt generiert und alle Projektmanager darüber per E-Mail informiert, wenn die Projekt-Checkbox angehakt ist.

Wird ein Auftrag gelöscht, bleiben zugehörige Mutterprojekte davon unberührt. Falls das Mutterprojekt allerdings bisher noch nicht in andere Projekte unterteilt wurde, wird es ebenfalls gelöscht.

Wunschbearbeiter AN:

Freies Textfeld zum Anfordern einer Wunschperson beim Auftragnehmer (AN) zum Bearbeiten des Auftrages.

Fertigstell.-Termin:

Spätestes geplantes Datum, bis zu welchem Termin der Auftrag bzw. der zugehörige Projekt abgeschlossen sein soll.

Projekt:

Durch setzen der Checkbox wird aus dem Auftrag gleich ein Projekt generiert.

Auftragsursache:

In diesem Feld steht wie der Name schon sagt die Ursache, der Grund für einen Auftrag. Dieses Feld ist ein Nachschlagewert und kann dort frei erweitert werden.

Auftragspriorität:

In diesem Feld steht wie der Name schon sagt die Priorität eines Auftrages. Dieses Feld ist ein Nachschlagewert und kann dort frei erweitert werden.

Auftragsziel:

Ausführliche Beschreibung des Auftragsziels.

Anmerkung:

In diesem Feld werden allgemeine Anmerkungen zu einem Projekt angezeigt.


Multi-Funktion

Durch Klick auf diesen Button werden die Änderungen durchgeführt.
