2FAs (2-Faktor-Authentisierungen) der einzelnen Benutzer verwalten


Die 2-Faktor-Authentisierungs-Verwaltung dient zum Erstellen, Ändern oder Löschen von 2FA-Geräte-ID's der Benutzer.




Benutzerauswahl

In dieser Maske können die Benutzerkriterien ausgewählt werden zu denen die Geräte-ID's für die 2FA verwaltet werden sollen. Der Suchtext und die ausgewählten Benutzer werden mit "ODER" verknüpft, d.h. wenn eine der beiden Bedingungen zutrifft wird der Benutzer aufgelistet.



Feldbeschreibungen

Benutzername:

In diesem Auswahlfeld können ein, mehrere oder alle Benutzer selektiert werden für welche die Geräte-ID's verwaltet werden sollen.

Suchtext:

Zeichenkette, nach der gesucht werden soll. Dies kann sowohl ein Teil des Benutzer- oder Personennamens sein als auch ein Teil der Geräte-ID.


Auswählen

Nach einem Klick auf diesen Button werden die Benutzer mit den entsprechenden Kriterien aufgelistet.




2FA-Geräte-ID's

2FA-Geräte-ID's der Benutzer



Feldbeschreibungen

Benutzername:

In dieser Spalte steht der Benutzername - die zugehörige Person steht als Überschrift in der übergeordneten Zeile.

?:

Ampel-Icon, dessen Tooltip Informationen bzgl. der Geräte-ID für diesen Benutzer anzeigt, d.h. ob eine Geräte-ID gespeichert ist oder ob 2FA aktiviert ist oder welche Auswirkungen eine Änderung einer Geräte-ID hat etc.

2FA-Geräte-ID:

Geräte-ID für die 2FA (2-Faktor-Authentisierung) eines Benutzers.

Geändert von:

In dieser Spalte steht wer wann zuletzt Änderungen an diesem Eintrag durchgeführt hat.

Löschen:

Mit einem Klick auf dieses Lösch-Icon wird die Geräte-ID für diesen und evtl. auch für anderen Benutzern mit derselben Geräte-ID entfernt. Benutzer mit aktivierter 2FA aber ohne zugewiesener Geräte-ID müssen ihr Gerät bei der nächsten Anmeldung registrieren.


Speichern

Durch Klick auf diesen Button werden die Änderungen durchgeführt, d.h. neue Geräte-ID's zugewiesen oder abgeändert.
