Projektkategorien verwalten

Verwalten von Projektkategoriegruppen und Projektkategorien.

Mit einem Klick auf das "Neue Projektkategoriegruppe erstellen"-Icon links unten in der Spalte "Prj.-Kategoriegruppe" wird eine neue Gruppe dynamisch mittels JavaScript in die Maske eingebunden. Jede Gruppe besteht aus einer änderbaren Sortiernummer, sowie einem änderbaren Namen und beliebig vielen (oder auch keinen) untergeordneten Projektkategorien.

Mit einem Klick auf das "Hinzufügen"-Icon in der Spalte "Projektkategorien" können beliebig viele Kategorien zu einer Gruppe dynamisch erstellt werden. Eine Kategorie besteht ebenfalls aus einer änderbaren Sortiernummer, sowie einem änderbaren Namen. Diese Kategorien können in den einzelnen Projekt-Programmfunktionen den Projekten zugeordnet werden.

Mit einem Klick auf das "Löschen"-Icon rechts neben jedem Gruppen- bzw. Kategorienfeld können diese gelöscht werden. Ist eine Kategorie bereits mindestens einem Projekt zugeordnet oder ist die Gruppe bzw. die Kategorie eine Standardgruppe bzw. -kategorie von fx-project, dann ist dieses Icon inaktiv und die Kategorie kann nicht gelöscht werden. Wird eine Gruppe gelöscht welche ein oder mehrere nicht löschbare Kategorien enthält, dann werden diese Kategorien zur Gruppe "Ohne Zuordnung" verschoben, bevor die gewählte Gruppe gelöscht wird.

Mit einem Klick auf das "Verschiebe"-Icon links vor jeder Kategorienzeile kann die Zuordnung zu den einzelnen Gruppen verändert werden. Existiert nur eine andere Gruppe, dann wird die Kategorie sofort verschoben, ansonsten muss die Zielgruppe in einem Popup-Fenster ausgewählt werden.

Vorgenommene Neuanlagen, Änderungen oder Löschungen werden in fx-project erst übernommen, wenn man auf den Button [Speichern] klickt.
