Festlegen von Terminen inkl. Einladen von Personen mit automatischer Mitteilung per E-Mail


In dieser Übersicht ist die komfortable Eingabe von eigenen oder fremden Terminen möglich. Der Termin kann dabei folgenden Kategorien angehören:

  • Einladung
  • Geschäftsessen
  • Konferenz
  • Kontakt
  • Meeting
  • Mitteilung
  • Privattermin
  • Verschiedenes

Für jeden Termin muss eine zuständige Person, sowie der Zeitraum angegeben werden. Wiederholungstermine können ebenfalls hier angelegt bzw. bearbeitet werden.

Zu jeder Terminart kann man beliebige Personen einladen, welche sofort per E-Mail benachrichtigt werden. Solche Einladungstermine können im Kontrollzentrum bestätigt oder abgelehnt werden.




Auswahl

Auswahl eines Termins zum Anzeigen, Ändern oder Löschen.

Personenschema:
Eigene Termine werden immer an oberster Stelle angezeigt. Der eigene Personenname ist dabei fett hinterlegt. Evtl. Termine anderer Personen werden je nach Berechtigung darunter in alphabetischer Reihenfolge aufgelistet.

Aufbau einer Terminzeile in der Liste:

  • Zuerst wird der Zeitraum des Termins angezeigt.
  • Betrifft der Termin einem selbst oder hat man das Recht den Termin zu ändern wird anschließend ein : Doppelpunkt gefolgt von der Terminart angezeigt. Ist eine nähere Beschreibung zu solchen Terminen vorhanden wird diese ebenfalls anschließend in Klammern angezeigt.
  • Ist der Termin ein Wiederholungstermin wird {+} am Ende angezeigt.
  • Ist der Termin ein Erinnerungstermin wird {!} am Ende angezeigt.
  • Ist der Termin bereits als "erledigt" gekennzeichnet beginnt die ganze Zeile mit einem ! Ausrufezeichen.
  • Falls der Termin nicht geändert werden kann, z.B. wegen fehlender Berechtigung, wird die ganze Zeile eingeschoben in Klammern angezeigt. Dies betrifft vor allem den Anzeigemodus.


HINWEIS:
Im Anzeigemodus werden bereits als "erledigt" gekennzeichnete Termine immer angezeigt, sonst nur falls im Zeitfilter eine Zeiteinschränkung gesetzt worden ist.

Termindaten

Termindaten

Die Auswahl im Feld Person ist abhängig davon, welche Berechtigung der User hat. Normale Mitarbeiter haben nur sich selbst, Vorgesetzte und Projektleiter alle Personen zur Auswahl.
Können Sie hier andere Personen wählen, so können Sie für diese Personen Termine anlegen.

Das direkte Anlegen von Urlauben ist nicht möglich, da alle Mitarbeiter einen Urlaubsantrag anzulegen haben, der dann vom Vorgesetzen oder der Verwaltung bestätigt werden muss.
Ferner müssen Urlaubstage immer halb- oder ganztägig sein.

Um andere Mitarbeiter zu Terminen einzuladen, müssen Sie die einzuladenden Personen in der folgenden Listenansicht mit einem Haken versehen. Wenn Sie dann auf einladen klicken, wird ein freier Termin gesucht, ein weiterer Klick auf Termin bestätigen legt die Termine an.

Bei Projektmitteilungen, die den Abschluss einer Aufgabe signalisieren, können Sie entscheiden, ob das Projekt abgeschlossen (Projektende) bleiben soll, oder nachgebessert (Nachbesserung) werden muss. In beiden Fällen erhalten die Projektmitarbeiter eine Mitteilung mit dem Text, den Sie im Feld Bestätigungsanmerk. angegeben haben..

HINWEIS:
Wiederholungstermine können angelegt werden, indem Sie eine Wiederholungsart und ein Enddatum der Wiederholung angeben. Nachdem Sie auf "OK" geklickt haben, wird der Termin ihren Spezifikationen nach angelegt.



Feldbeschreibungen

Defaultaktion:

Hier sollte normalerweise "Anzeigen" eingetragen werden.

Person:

In diesem Auswahlfeld wird die Person gewählt, deren Daten je nach Aktion bearbeitet oder angezeigt werden soll.

Terminart:

Das Feld "Terminart" stellt verschiedene Terminarten zur Auswahl. Während Projektleiter alle geschäftlichen Termine eines Mitarbeiters einsehen können, bleibt Ihnen die Kontrolle von privaten Terminen untersagt.

Gep. Startdatum:

Hier wird das Soll-Startdatum des Projektes angezeigt bzw. angegeben, d.h. ab wann laut Planung mit dem Projekt zu beginnen ist.

Terminstart:

Gibt die Startzeit für einen Termin an.

Wiederholung:

Gibt die Wiederholung eines Termins an (täglich, wöchentlich etc.).

Ende Wdh.:

Gibt an, an welchem Tag die Wiederholung eines Termins endet.

Bsp.: Am 13.2. wird ein Termin mit Wiederholung angelegt. Ist bei der Terminanlage z.B. "täglich" gewählt, und das Enddatum der Wiederholung der 20.2., so wird dieser Termin acht Mal eingetragen, und zwar zwischen dem 13. bis 20. Februar

Erinnerung:

Datumsfeld für die Erinnerung an einen Termins.

Wird ein Datum eingetragen, so wird an diesem Tag ein Erinnerungstermin an den momentan aktuellen Termin angelegt.

Terminende:

Gibt das Ende eines Termins.

Dauer:

Gibt an, ob sich ein Termin über den ganzen Tag erstreckt, oder nur halbtägig. Dabei ist die jeweilige Dauer von der Definition des Arbeitstages abhängig, z.B. von 8 Uhr bis 17 Uhr entspräche einem ganztägigen Termin bei einem Standard Acht-Stunden-Tag mit einer Stunde Pause.

Bleibt dieses Feld leer werden die von Ihnen eingegebenen Uhrzeitangaben herangezogen.

Anzahl Wdh.:

Anzahl der Wiederholungen

Hier kann alternativ zum Enddatum angegeben werde wie oft ein Termin wiederholt werden soll.

Projekt:

Im Feld Projekt werden die Vorgänge (Projekte oder Aufgaben) aufgelistet, für die man die entsprechenden Rechte besitzt und die entsprechen Filterkriterien gelten.

Zeiterf.-Termin:

Ein Zeiterfassungstermin ist ein Termin, welcher im Kontrollzentrum aufgelistet wird und durch einen einfachen Klick in die Zeiterfassung übernommen werden kann.

Ist ein Zeiterfassungstermin einem Projekt/Teilprojekt zugeordnet, dann wird die Zeiterfassung dem nächsten nachfolgenden Terminprojekt zugebucht. Ist der Termin hingegen keinem Projekt zugeordnet, dann wird auf [Unbekannt] gebucht.
Anwendungsbeispiele sind Einladungstermine zu Projektbesprechungen, Geschäftsessen etc.

Treffpunkt:

Dieses Feld gibt den Treffpunkt eines Termins an (soll nur als Hilfe dienen).

Erledigt:

Ist diese Box abgehakt, heißt das nicht, dass der Termin gelöscht wird, es wird lediglich als "erledigt" gekennzeichnet.

Betreff:

Gibt den Betreff eines Termins an.

Beschreibung:

Erläuternde Beschreibungen, Detailangaben, Bemerkungen oder ähnliches.


Multi-Funktion

Durch Klick auf diesen Button werden die Änderungen durchgeführt.

Eingaben prüfen
