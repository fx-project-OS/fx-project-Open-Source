Administration > Benutzer > Passwort

Ändern des individuellen Passworts zum Anmelden in fx-project®


Das Passwort sollte aus Sicherheitsgründen regelmäßig geändert werden. Passwörter sollten nicht aus Namen, Geburtstagen oder anderen bekannten Informationen zu einem Benutzer sein, sondern willkürlich aus Ziffern und Buchstaben bestehen.




Passwort ändern und Benutzer aktivieren

Hier können Sie Ihr Passwort ändern.

Aus Sicherheitsgründen muss dazu in der obersten Zeile das alte Passwort eingegeben werden, sowie in den beiden unteren Zeilen das neue Passwort.

HINWEIS:
Durch eine Passwortänderung wird der Benutzer automatisch aktiviert, falls dieser gesperrt oder inaktiv war!



Feldbeschreibungen

Altes Passwort:

In dieses Feld muss das "alte" Passwort aus Sicherheitsgründen eingegeben werden.

Neues Passwort:

In dieses Feld muss ein persönliches Passwort eingegeben werden. Das Passwort sollte zur eigenen Sicherheit mindestens sechs Zeichen lang sein, kein "$"-Sonderzeichen etc. enthalten, sowie aus keinen leicht zu erratenden Buchstaben- bzw. Ziffernfolgen bestehen wie "123456", "654321", "qwertz" etc.

Das Passwort wird intern verschlüsselt gespeichert, so dass bei einem eventuellen "Vergessen" ein neues Passwort beantragt werden muss.

Passwort wiederholen:

Bitte geben Sie das neue Passwort nochmal ein, damit Tippfehler ausgeschlossen werden können.


Ändern

Ein Klick auf diesen Button ändert das Passwort endgültig ab.
