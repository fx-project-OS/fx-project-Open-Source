Auswerten von Kommunikationdaten bei Personen


Der Personen-Kommunikationsreport ist ein detaillierter Bericht über die unterschiedlichsten Kommunikationsdaten von Personen. Dabei kann der frei definierbare Kommunikationstyp ebenso eingestellt werden, wie ein Suchtext, Postleitzahl, Länderkennzeichen oder Personentyp, zur Eingrenzung der gefundenen Daten. Durch die integrierte Rechtevergabe ist gewährleistet. dass jede Person nur die für sie vorgesehenen Daten aufgelistet bekommt.




Auswahlkriterien festlegen

Hier können die Auswahlkriterien für den Personen-Kommunikationsreport festgelegt werden.



Feldbeschreibungen

Kommunikationstyp:

In diesem Feld wird aus einem Auswahlmenü die Art der Kommunikationsverbindung, wie z.B. Telefon, Fax, E-Mail etc. ausgewählt.

Zugeordnet zu Objekt:

In diesem Feld steht die "übergeordnete Einheit".
Bei Projekten wäre dies das übergeordnete Projekt [(Teil-)Projekt].

Bei Dokumenten wäre dies die Einheit, der Sie angehören (z.B. Projekt, Termin, Notiz etc.).

Suchtext:

In diesem Feld wird die Zeichenkette eingegeben, nach der gesucht werden soll.

Text:

Postleitzahl

Text:

Postleitzahl: Von:

Text:

Postleitzahl: Bis:

Länderkennzeichen:

In diesem Feld wird ein Länderkennzeichen aus einer Ländertabelle ausgewählt.

Das Länderkennzeichen entspricht dabei den aktuell gültigen Autokennzeichen, z.B. "USA", "F", "NL", "D" etc.

Anmerkung anzeigen:

Hier wird festgelegt ob die Anmerkung angezeigt werden soll oder nicht. Das Markieren dieser Checkbox bedeutet, dass die Anmerkungs-Spalte im Report mit angezeigt werden soll, sonst wird sie ausgeblendet.

inaktiv:

Durch setzen der Checkbox werden auch inaktive Personen (Mitarbeiter, Auftraggeber, usw.) angezeigt.

Personen-Typ:

Kategorisierung der Personen, zu unterscheiden sind z.B. Mandanten, Mitarbeiter, Kunden etc.

Sortierreihenfolge:

Festlegen der Sortierreihenfolge der angezeigten Felder.


Anzeigen

Durch Klick auf diesen Button wird Liste erzeugt und angezeigt. Je nach festgelegten Auswahlkriterien wird die Liste hierbei entsprechend gefiltert.




Report

Report der ausgewählten Kommunikationsdaten.



Feldbeschreibungen

Nummer:

In diesem Feld wird die eigentliche Kommunikationsverbindung, wie z.B. Telefonnummer, Faxnummer, E-Mail-Adresse etc. eingetragen bzw. angezeigt. Die Eingaben in diesem Feld sind abhängig vom vorher ausgewählten Kommunikationstyp.

Anmerkung:

In diesem Feld werden allgemeine Anmerkungen zu einem Projekt angezeigt.

Name:

In diesem Feld wird der Familienname der Person oder die Bezeichnung der juristischen Person (z.B. Firmenname) angezeigt bzw. eingetragen.

Adresse:

Es handelt sich hier um einen internen Schlüssel zur eindeutigen Identifikation von Adressen.

Ansprechp.:

Dieses Feld zeigt den Ansprechpartner bzw. den Kunden. Durch Selektion eines Ansprechpartners wird in einigen Fällen (z.B. Projektdefinition) automatisch auch ein Kunde gewählt, da jeder Ansprechpartner genau einem Kunden zugeordnet ist.

Angelegt wird ein Ansprechpartner über die Eingabemaske im Menüpunkt "Stammdaten".

Nummer:

Gibt die Identifikationsnummer des Kommunikationseintrages an.

Geänd. von:

Gibt an, welcher Benutzer zuletzt einen Datensatz bearbeitet hat.
