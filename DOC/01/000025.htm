Administration > Benutzer > Benutzereinstellungen

Diverse Einstellungsmöglichkeiten die von jedem Benutzer getätigt werden können


Individuelle Einstellungen, welche nur für den eigenen Benutzer zutreffen.




Einstellungen

Hier können verschiedene Einstellungen nach persönlichen Vorlieben abgeändert werden, z.B. das "Datumsformat", das "Dezimaltrennzeichen", die "Anzahl der Nachkommastellen", die "Anzahl der Listeneinträge", die "Sortierreihenfolge von Projekten" usw.

Ausgegraute Einstellungen wurden noch nicht angepasst, d.h. diese enthalten die festgelegten ursprünglichen Standardwerte Ihres Mandanten.

Bereits angepasste Benutzereinstellungen können wieder auf den ursprünglichen Standardwert zurückgesetzt werden durch Anhaken der "Löschen"-Checkbox vor dem entsprechenden Einstellungswert.

Durch einen Klick auf das "Fragezeichen" am Zeilenende wird für den entsprechenden Einstellungswert ein eigenes Hilfefenster mit einer exakten Beschreibung der Änderungsmöglichkeiten geöffnet.



Feldbeschreibungen

Löschen-Checkbox:

Checkbox zum Markieren der bereits angepassten Benutzereinstellung, welche wieder auf den ursprünglichen Standardwert zurückgesetzt werden soll.

Text:

Textfeld

Variables Textfeld, welches programmgesteuert angepasst werden kann.

Einstellung:

Kurzbeschreibung der Einstellung. Ausgeraute Einstellungen wurden noch nicht verändert, d.h. diese besitzen noch den ursprünglichen Standardwert.

Einstellungswert:

Gesetzter Wert der Einstellung. Dieser Wert kann nach persönlichen Vorlieben abgeändert werden.

Fragezeichen:

Durch einen Klick auf dieses Icon wird für den entsprechenden Einstellungswert ein eigenes Hilfefenster mit einer exakten Beschreibung der Änderungsmöglichkeiten geöffnet.


Speichern

Ein Klick auf diesen Button speichert die Einstellungen ab.
