Systemanforderungen

Systemanforderungen für fx-project sind ein unterstütztes Betriebssystem, die Interpretersprache PHP, ein Webserver, ein Datenbanksystem sowie eine Netzwerkkarte.

Hier die genaue Beschreibung der Anforderungen:


SERVER

Betriebssystem:Windows Server 2012 oder höher, Windows 7 oder höher, Linux oder Unix
Webserver:IIS-Server: 7 oder höher, Apache: 2.4 oder höher (andere auf Anfrage)
DB-System:PostgreSQL: 8 oder höher, SQL Server: 2012 oder höher (andere auf Anfrage)
Script-Interpreter:PHP ab 8.3 inkl. IonCube PHP Loader für Sonderprogramme
Netzwerkprotokoll:TCP/IP (SSL optional)
Netzwerkkarte:Standard

CLIENT

Webbrowser:mind. Internet Edge, Firefox, Opera, Google Chrome (HTML 5.0-kompatibler Browser, JavaScript erforderlich)
