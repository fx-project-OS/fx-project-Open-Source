2 F A
(2-FAKTOR-AUTHENTSIERUNG)  













VERSION 24.1  
  

Registrierung

Ist noch kein Gerät (PC / Smartphone / Tablet etc.) für die 2FA hinterlegt erscheint nach dem Anmelden folgender Anleitungstext.


Der QR-Code auf Ihrer Anmeldeseite verweist hierbei auf die 2FA-Browser-App-Webseite von Ihrem System. Scannen Sie nun mit Ihrem Gerät den QR-Code und öffnen Sie diesen Link in Ihrem Browser des entspr. Gerätes. Als Parameter werden übermittelt die Sprache, Mandanten-Nummer, sowie der Benutzername.


In der Browser-App muss der bei Ihnen angezeigte Registrierungscode eingegeben werden.


Dieser wird nach einem Klick auf den Button [Registrieren] zusammen mit der eindeutigen "Geräte-ID" übermittelt und bei Übereinstimmung beim jeweiligen Benutzer hinterlegt. In der App wird daraufhin eine Erfolgsmeldung angezeigt und Sie werden automatisch zum nächsten Punkt "Anmeldung" weitergeleitet.

Anmeldung

Ist die 2FA beim Mandanten freigeschaltet und beim Benutzer bereits ein Gerät (PC / Smartphone / Tablet etc.) hinterlegt erscheint nach dem Anmelden oder nach erfolgreicher Registrierung folgender Anleitungstext.


Der QR-Code auf Ihrer Anmeldeseite verweist hierbei auf die 2FA-Browser-App-Webseite von Ihrem System. Scannen Sie nun mit Ihrem Gerät den QR-Code und öffnen Sie diesen Link in Ihrem Browser des entspr. Gerätes. Als Parameter werden übermittelt die Sprache, Mandanten-Nummer, sowie der Benutzername.


In der Browser-App wird nun ein Token Code angezeigt, welches beim Erstaufruf oder nach 60 Sekunden in Synchronisation mit fx-project erzeugt wurde/wird. Der Zeitbalken oben in der Browser-App zeigt an wieviel Zeit noch verbleibt bevor ein neues Token generiert wird.


Dieser Token Code muss in der Anmeldemaske von fx-project eingegeben werden und bei Übereinstimmung wird die Anmeldung als "gültig" eingestuft und fx-project geladen.


Information
