Workflow-Menü (Workflow-Slider)

Das Workflow-Menü, oder auch Workflow-Slider genannt, befindet sich auf der linken Bildschirmseite und dient zum optisch unterstützen Aufrufen der Programmfunktionen.

Der Workflow-Slider ist das obere der beiden Slider-Elemente auf der linken Seite.

Nähert man sich mit dem Mauscursor wird die dezente "Lasche" größer und beschriftet. Mit einem Klick auf diese "Lasche" kann das Workflow-Menü auf- und zugeklappt werden.

Geöffnet erhält der Benutzer eine Übersicht über alle Hauptabschnitte berechtigter Programmfunktionen.



Innerhalb des Workflow-Menüs hat der Benutzer folgende Steuerungsmöglichkeiten:

Schwarz-Weiß-Anzeige → Umschalten auf farbige Anzeige
Farbige Anzeige → Umschalten auf Schwarz-Weiß-Anzeige


Durch Klicken auf die jeweiligen Menüpunkte kann man sich bis zu den gewünschten Programmfunktionen durchnavigieren.



Für die jeweilge Aktion stehen ggf. folgende Möglichkeiten zur Verfügung:

Programmfunktion aufrufen im Modus: Anzeigen
Programmfunktion aufrufen im Modus: Neuanlage
Programmfunktion aufrufen im Modus: Ändern
Programmfunktion aufrufen im Modus: Löschen
