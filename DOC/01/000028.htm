Auflistung der zugeordneten Mitarbeiter eines Projekts, Teilprojekts oder Aufgabe mit Auslastung


Auflistung, welche Mitarbeiter zu welchen Aufgaben zugeteilt sind.




Dient zur Auswahl von Ressourcen

Dient zur Auswahl von Mitarbeitern

In Abhängigkeit von den Angaben, die Sie in den Feldern Person, Projekt, Zeitspanne bis und Zeitspanne bis gemacht werden, wird ein Report angezeigt, welche Mitarbeiter des gewählten Projekts im gewünschten Zeitraum geplant ist. Werden die Felder nicht ausgefüllt, werden alle Mitarbeiter aller Projekte, unabhängig vom Zeitraum gelistet.

HINWEIS:
Beim ersten Aufruf, bzw. bei Klick auf das Personen-Icon, kann der Personenkreis des Personenauswahlfeldes eingeschränkt werden, z.B. nur "interne Mitarbeiter".



Feldbeschreibungen

Projekt:

Im Feld Projekt werden die Vorgänge (Projekte oder Aufgaben) aufgelistet, für die man die entsprechenden Rechte besitzt und die entsprechen Filterkriterien gelten.

Person:

In diesem Auswahlfeld wird die Person gewählt, deren Daten je nach Aktion bearbeitet oder angezeigt werden soll.

Interner Schlüssel für die Tabelle Material:

Interner Schlüssel für die Tabelle Material.

Materialtyp:

Typ eines Materials, d.h. zu welcher Materialgruppe dieses gehört. Die Materialtypen können als Nachschlagewerte selbst definiert werden.

Materialart:

Mittels diesem Feld können Sie angeben, ob Sie nur die "einmaligen Materialien" oder die "Ressourcen Material" in diesem Report sehen wollen.

Entsprechend Ihrer Auswahl wird daraufhin der entsprechende Report angezeigt.

Einmaliges Material: Material deren Anschaffungskosten auf das gewählte Projekt aufaddiert werden, die aber nicht als ("arbeitende") Ressourcen im Projekt geplant sind (z.B. Werkstoffe zum Herstellen bestimmter Güter).

Derartige Materialien müssen nur in der Programmfunktion Erfassung>Materialien>Materialdefinition definiert werden.

Ressourcen Material: Materialien, die als Ressourcen in einem Projekt "arbeiten" (z.B. ein Kran, der 8 Stunden pro Tag auf einer Baustelle eingesetzt wird).

Diese Materialien müssen in der Programmfunktion Erfassung>Materialien>Materialdefinition definiert werden, außerdem müssen Sie über die Programmfunktion Durchführung>Material>Bestand als Ressource definiert und in Durchführung>Material>Verbrauch einem Projekt zugeteilt werden.

Zeitraum von:

In diesem Feld wird angezeigt, von welchem Zeitpunkt an Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Zeitraum bis:

In diesem Feld wird angezeigt, bis zu welchem Zeitpunkt die Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Feldbeschreibungen

Name:

In diesem Feld wird der Familienname der Person oder die Bezeichnung der juristischen Person (z.B. Firmenname) angezeigt bzw. eingetragen.

Materialname:

Name oder Bezeichnung eines Materials.

Materialtyp:

Typ eines Materials, d.h. zu welcher Materialgruppe dieses gehört. Die Materialtypen können als Nachschlagewerte selbst definiert werden.

Gruppe:

In diesem Feld werden die einzelnen Fähigkeitsgruppen der Mitarbeiter angezeigt, d.h. die Mitarbeiterbereiche, die diese abdecken.

Fähigkeit:

In diesem Feld stehen die Bezeichnungen der Fähigkeiten der einzelnen Personen, z.B. "Englisch", "Russisch" o.ä. als Fähigkeiten für die Gruppe "Fremdsprachen".

Start:

Dieses Feld gibt das Plan-Startdatum an. Beim Anlegen ist es identisch mit dem Datum des Projektbeginns, zu dem der Mitarbeiter zugeteilt wurde, es kann aber nachträglich geändert werden.

Ende:

Dieses Feld gibt das Plan-Enddatum an. Beim Anlegen ist es identisch mit dem Datum des Projektendes, zu dem der Mitarbeiter zugeteilt wurde, es kann aber nachträglich geändert werden.

EK:

Geldwert, dessen Höhe den finanziellen Aufwand für das Unternehmen pro Einsatzstunde eines Arbeiters oder eines Materials zeigt (Gegenstück zu VK=Verkaufspreis). Hier ist der EK (Einkaufspreis) einzugeben, den die Ressource für das momentane Projekt erhält.

Beispiel: Ein Mitarbeiter hat einen EK von 25 €. Arbeitet er an einem "normalen" Projekt ist sein EK auch auf 25 €. Arbeitet er aber an einem Projekt, das all seine Erfahrung benötigt kann der EK auf dieses Projekt angepasst werden (z. B. EK 40 €). Die Anpassung ist beim VK ebenso möglich.

VK:

Geldwert, dessen Höhe den finanziellen Ertrag für das Unternehmen pro Einsatztag eines Arbeiters oder eines Materials zeigt (Gegenstück zu EK=Einkaufspreis). Hier ist der VK (Verkaufspreis) anzugeben, den die Ressource für das momentane Projekt erwirtschaftet.

Beispiel: Ein Mitarbeiter hat einen VK von 40 €. Arbeitet er an einem "normalen" Projekt ist sein VK auch auf 40 €, arbeitet er aber an einem Projekt, das all seine Erfahrung benötigt kann der VK auf dieses Projekt angepasst werden (z. B. VK 60 €). Die Anpassung ist beim EK ebenso möglich.

Aufw.-Std.:

Gibt an, wieviel Stunden ein Mitarbeiter für eine Aufgabe aufwenden soll.

%:

Prozent (%)

Auslastung:

Gibt den Prozentsatz an, zu dem eine Ressource bereits (durch andere Projekte) ausgelastet ist.

Material haben bei Zuteilung immer 100% Auslastung

Bei Mitarbeitern errechnet sich die Auslastung aus den Werten des Soll-Arbeitstags und des Soll-Aufwands für die Aufgabe.
Bsp.: Ein Mitarbeiter mit einem Standard Acht-Stunden-Tag wird einer Aufgabe mit dem Soll-Aufwand von 24 Stunden zugeteilt. Die Laufzeit der Aufgabe beträgt drei Tage.
=> Innerhalb dieser drei Tage kann der Mitarbeiter laut Soll-Arbeitszeit drei mal acht Stunden arbeiten
=> 24 Stunden / (3 * 8 Stunden) => 1 => Auslastung des Mitarbeiters beträgt genau 100% für diese Aufgabe.
Nun nehmen wir an dieselbe Aufgabe ist nun in vier Tagen zu erledigen
=>24 Stunden / (4 * 8 Stunden) => 0.75 => Auslastung des Mitarbeiters beträgt genau 75% für diese Aufgabe
Wenn mehrere Mitarbeiter einer Aufgabe zugeteilt werden, so kann der Aufwand aufgeteilt werden, dass z.B. ein Mitarbeiter 16 Stunden leistet, der andere 8 Stunden. Bei einem Standard Acht-Stunden-Tag und einer Aufgaben-Laufzeit von vier Tagen würden sich die Auslastungen der beiden Mitarbeiter in dem eben genannten Fall auf 50% (=> 32 / 16) bzw. 25% (=> 32 / 8) belaufen.

Die Gesamtauslastung eines Mitarbeiters zu einem Termin errechnet sich aus den einzelnen Auslastungen innerhalb der Aufgaben zu diesem Termin, d.h. wenn ein Mitarbeiter an einem Tag drei Aufgaben zugeteilt hat, mit einzelnen Auslastungen von 40%, 30% und 20%, dann ergäbe sich für diesen Tag eine Gesamtauslastung für diesen Mitarbeiter von 90%.

Fehl- und Feiertage werden bei der Auslastungsberechnung berücksichtigt
