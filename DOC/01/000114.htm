Reporting > Zeitdaten > Leistungsnachweis

Auflistung der Zeiterfassung eines bestimmten Zeitraums mit diversen Selektionskriterien (Projekt, Auftraggeber, Mitarbeiter etc.)


Der Leistungsnachweis gilt in erster Linie zur Vorlage der erbrachten Leistungen für den Auftraggeber in einem bestimmten Zeitraum. Über den Dateimanager wird ein druckfähiger Report mit allen Daten abzeichnungsfertig erstellt.




Anzeigekriterien für den Leistungsnachweis einstellen.

Anzeigekriterien für den Leistungsnachweis einstellen.

Festlegung von Auswahlkriterien, nach denen der Leistungsnachweis der einzelnen Projekte für den entsprechenden Mitarbeiter aufbereitet werden soll. Hierbei kann insbesondere die darzustellende Zeitspanne eingestellt werden, sowie das zugrundeliegende Projekt.

Bei Anzeigeart "Automatisch" wählt das Programm eigenständig die aufzuzeigende Zeitspanne aus.



Feldbeschreibungen

Zeitspanne von:

In diesem Feld wird angezeigt, von welchem Zeitpunkt an Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Projekt:

Im Feld Projekt werden die Vorgänge (Projekte oder Aufgaben) aufgelistet, für die man die entsprechenden Rechte besitzt und die entsprechen Filterkriterien gelten.

Zeitspanne bis:

In diesem Feld wird angezeigt, bis zu welchem Zeitpunkt die Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Auftraggeber:

Hier wird der Auftraggeber des Vorgangs (Projekt, Teilprojekt bzw. Aufgabe) gewählt.

Max. Zeichen.:

Festlegung der Anzahl der maximal darzustellenden Textzeichen, d.h. ist ein Text zu lang (insbesondere für die Druckaufbereitung) wird der Text ab der eingegebenen Anzahl an Zeichen umgebrochen oder abgeschnitten, je nach gesetztem Feld "Zeilenumbruch".

Eine Eingabe von '0' bedeutet: Kein Textumbruch. Für die Druckaufbereitung z.B. beim "Leistungsnachweis" empfiehlt sich der voreingestellte Wert von 50 (= Standard).

Person:

In diesem Auswahlfeld wird die Person gewählt, deren Daten je nach Aktion bearbeitet oder angezeigt werden soll.

Max. Projektz.:

Festlegung der Anzahl der maximal darzustellenden Textzeichen für die Projektbezeichnung, d.h. ist ein Text zu lang (insbesondere für die Druckaufbereitung) wird der Text ab der eingegebenen Anzahl an Zeichen umgebrochen oder abgeschnitten, je nach gesetztem Feld "Zeilenumbruch". Bei Eingabe eines Wertes wird außerdem das Projekt ohne die Projektnummerierung dargestellt.

Eine Eingabe von '0' bedeutet: Kein Textumbruch und Projektnummerierung (= Standard).

Anzeigeart:

Legt fest, wie bestimmte Daten innerhalb eines Reports aufbereitet werden sollen.

Zeilenumbruch:

Festlegung, ob Texte nach einer gewissen Zeichenanzahl umgebrochen oder abgeschnitten werden sollen. Bei einem Textumbruch wird der nach einer gewissen Zeichenanzahl verbleibende Text jeweils in der nächsten Zeile fortgesetzt und evtl. erneut umgebrochen usw.

Zwischensumme:

Durch anhaken dieser Checkbox kann festgelegt werden, ob Zwischensummen angezeigt werden sollen oder nicht.

Beachten Sie: Im Tagesmodus werden Zwischensummen nach jeder Woche gebildet, falls die Checkbox angehakt ist.

Leereinträge:

Durch an- bzw. aushaken dieses Feldes kann festgelegt werden, ob nicht relevante Einträge (=Leereinträge) angezeigt werden sollen oder nicht.

Wird z.B. ein Monat für ein Projekt aufgelistet, jedoch nur an zwei Tagen in diesem Monat wurde auch für dieses Projekt gearbeitet, so kann hier entschieden werden ob der gesamte Monat oder nur diese beiden Tage angezeigt werden sollen.

Projekt-Nr.:

Dies ist die Verzeichnisnummer für ein Projekt.

Die Verzeichnisnummer kann bei Mutterprojekten nicht geändert werden.

Die Verzeichnisnummer wird beim Löschen oder Verschieben automatisch angepasst.

Durch manuelles Ändern der Verzeichnisnummer können Sie Projekte innerhalb einer Hierarchie verschieben.

Reisekosten:

Nur Anzeigefunktion: Literal für den Reisekosten-Block in der Zeiterfassung.

Unbekannt:

Falls diese Checkbox angehakt ist werden auch "unbekannte" Aufgaben angezeigt.

Freigabe:

Die Freigabe zeigt an, ob der Datensatz zur Abrechnung freigegeben wurde.


Anzeigen

Durch Klick auf diesen Button wird Liste erzeugt und angezeigt. Je nach festgelegten Auswahlkriterien wird die Liste hierbei entsprechend gefiltert.




Leistungsnachweis

Leistungsnachweis

Hier wird der Leistungsnachweis für die einzelnen Mitarbeiter, je nach eingestellter Zeitspanne, angezeigt. Als Basisprojekt dient dabei das im Auswahlfeld eingestellte Projekt bzw. Teilprojekt.

Alle diesem Projekt zugeordneten Teilprojekte bzw. Aufgaben werden im Report aufgelistet, sofern dazu eine Stundenaufstellung existiert.



Feldbeschreibungen

Tag:

Vollständige Bezeichnung des Wochentags.

Datum:

Dient zur Eingabe oder Anzeige eines Tagesdatums. Bei einer erforderlichen Datumseingabe ist meistens der Standardwert (das heutige Datum) bereits vorbelegt.

Zeit:

Dient zur Eingabe oder Anzeige einer Uhrzeit.

Pause:

In Pause gibt der Benutzer die Summe seiner Pausen ein, die er während der geleisteten Tätigkeit eingelegt hat.

Projekt:

In diesem Feld werden Mitarbeiter angezeigt, die in das aktuelle Projekt eingebunden sind.

Leist.-Art:

Die Leistungsart in der Kostenrechnung ist eine Bezugsgröße, die die qualitative technische Beschreibung der Produktionstätigkeit angibt. Hierbei kann klassifiziert werden zwischen produktiven (=wertschöpfenden) und nicht-produktiven Arten.

Tätigkeit:

In diesem Feld erhält man die Möglichkeit seine geleisteten Arbeiten ausführlich zu beschreiben. Je exakter die Beschreibungen, desto genauer kann zu einem späteren Zeitpunkt der Arbeitsfortschritt wieder rekonstruiert werden.

Anmerkung:

In diesem Feld werden allgemeine Anmerkungen zu einem Projekt angezeigt.

Geänd. von:

Gibt an, welcher Benutzer zuletzt einen Datensatz bearbeitet hat.

Ges.-Std.:

In diesem Feld stehen die Gesamtstunden für eine eingegebene Tätigkeit, egal ob die geleistete Arbeit verrechenbar ist oder nicht.

Ver.-Std.:

In diesem Feld stehen die verrechenbaren Stunden für eine eingegebene Tätigkeit.
