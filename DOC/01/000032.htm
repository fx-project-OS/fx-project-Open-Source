Suchen und Auflisten von bestimmten Terminen


Komfortable Suche von Terminen nach miteinander kombinierbaren Kriterien. Diese Kriterien sind "Person", "Betreff", "Zeitraum" und "Terminart".
Alle gefundenen Termine werden übersichtlich untereinander nach Personen und Zeitraum aufgelistet. Falls die Berechtigung vorliegt, wird der angezeigte Termin mit einer Änderungsmöglichkeit verlinkt.




Suchkriterien

Suchkriterien

Suche nach bestimmten Terminen unter Mithilfe von Suchkriterien, z.B.
nach einem Teil des Betreffs des Termins oder
nach einer bestimmten Terminart.

Natürlich können Sie auch verschiedene Suchkriterien miteinander kombinieren.
Nach einem Klick auf "Suchen" wird nach den Terminen gesucht, welche die Kriterien erfüllen.



Feldbeschreibungen

Person:

In diesem Auswahlfeld wird die Person gewählt, deren Daten je nach Aktion bearbeitet oder angezeigt werden soll.

Betreff:

Gibt den Betreff eines Termins an.

Suche von:

In diesem Suchfeld können Sie den Gültigkeitsbeginn des Zeitrahmens der Suche festlegen.

Suche bis:

In diesem Suchfeld können Sie das Gültigkeitsende des Zeitrahmens der Suche festlegen.

Terminart:

Das Feld "Terminart" stellt verschiedene Terminarten zur Auswahl. Während Projektleiter alle geschäftlichen Termine eines Mitarbeiters einsehen können, bleibt Ihnen die Kontrolle von privaten Terminen untersagt.


Suchen

Durch Klick auf diesen Button wird die Suche gestartet.




Suchergebnisse

Suchergebnisse

Alle Treffer zu einer Suche werden in einer Liste angezeigt.
Mit einem Klick auf den Button OK (rechts neben jedem Treffer), wird man automatisch zu diesem Termin weitergeleitet.



Feldbeschreibungen

Terminstart:

In diesem Feld wird die Startzeit für einen Termin angezeigt.

Terminende:

In diesem Feld wird die Endzeit für einen Termin angezeigt.

Terminart:

Das Feld "Terminart" stellt verschiedene Terminarten zur Auswahl. Während Projektleiter alle geschäftlichen Termine eines Mitarbeiters einsehen können, bleibt Ihnen die Kontrolle von privaten Terminen untersagt.

Betreff:

Gibt den Betreff eines Termins an.

OK:

Icon zum Bestätigen der Aktion.
