Verwalten von mandantenweiten Feiertagen


Je Mandant können separat die entsprechenden Feiertage aktiviert werden. Feiertage können sowohl ein ganzer oder halber Arbeitstag sein, welche sich auf die Soll-Arbeitszeit der Mitarbeiter für die Zeiterfassung etc. auswirken. Zusätzliche, nicht erfasste Feiertage können definiert werden.





Feiertage für das Jahr / und für das Land

Hier können sämtliche Feiertage geändert, gelöscht oder komplett neu angelegt werden.

Wenn Sie (halbe) Brückentage anlegen, werden für die gesamte Belegschaft (halbe) Urlaubstage angelegt.

Voraussetzung dafür ist jedoch, dass

1. der Mitarbeiter an diesem Tag noch keinen Urlaubs- oder Krankheitstag eingetragen hat
2. der Mitarbeiter an diesem Tag einen Arbeitstag eingetragen hat
3. der Mitarbeiter an diesem Tag bereits einen definierten Arbeitsbeginn und eine Arbeitsdauer eingetragen hat
4. der Brückentag in der Zukunft liegt.

Für jeden angelegten (halben) Brückentag wird jedem Mitarbeiter ein (halber) Urlaubstag angerechnet.



Feldbeschreibungen

Chk.:

Aktivieren bzw. Deaktivieren eines Eintrages durch Setzen eines Häkchen in die "Checkbox"

Tag:

Vollständige Bezeichnung des Wochentags.

Datum:

Dient zur Eingabe oder Anzeige eines Tagesdatums. Bei einer erforderlichen Datumseingabe ist meistens der Standardwert (das heutige Datum) bereits vorbelegt.

Feiertag:

Gibt den Feiertagsname an.

Halb:

Gibt an, ob dieser Feiertag ganz- oder halbtägig oder nur zum Anzeigen bestimmt ist.


Chk.

Durch Klick auf diesen Button werden die Änderungen durchgeführt.
