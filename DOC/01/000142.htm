Verwalten des Jahresurlaubs und/oder Vorjahresurlaubsübertrags


Der Jahresurlaub jedes Internen Mitarbeiters kann hier komfortabel geändert, sowie ein Übertragswert aus dem Vorjahr gesetzt werden.




Personenauswahl

Auswahlkriterien festlegen zur Filterung der aufzulistenden Personen, für die die Jahresurlaube angezeigt und/oder festgelegt werden sollen.



Feldbeschreibungen

Suchtext:

In diesem Feld wird die Zeichenkette eingegeben, nach der gesucht werden soll.

Zeitspanne von:

In diesem Feld wird angezeigt, von welchem Zeitpunkt an Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Aktive Mitarbeiter:

Mit dieser Checkbox kann ausgewählt werden, ob alle Mitarbeiter angezeigt werden sollen, die den Auswahlkriterien entsprechen, oder nur die Mitarbeiter, die einen anzeigbaren aktiven Datensatz in diesem Zeitraum vorweisen.

Checkbox angehakt = Nur Mitarbeiter, die einen anzeigbaren aktiven Datensatz haben.

Abteilung:

Gibt die Abteilung an, in welcher der Mitarbeiter arbeitet.

Zeitspanne bis:

In diesem Feld wird angezeigt, bis zu welchem Zeitpunkt die Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Historie anzeigen:

Ist diese Checkbox angehakt wird neben den normalen Daten auch ein zusätzlicher Historienverlauf mit angezeigt.


Anzeigen

Durch Klick auf diesen Button wird Liste erzeugt und angezeigt. Je nach festgelegten Auswahlkriterien wird die Liste hierbei entsprechend gefiltert.




Vorjahresurlaub

Vorjahresurlaub

Eintragen des Urlaubs vom vorigen Jahr, wenn die Zusatzdaten der Person schon eingetragen sind.

Sie sehen alle Mitarbeiter des Mandanten mit dem jährlichen Urlaubsanspruch, dem derzeitigen Resturlaub, sowie dem Eingabefeld "aus Vorjahr".

Tragen Sie im Feld "aus Vorjahr" den jeweiligen Vorjahresurlaub der Person ein und wiederholen Sie diesen Vorgang, bis alle Personen, die noch über einen Vorjahresurlaub verfügen, bedacht wurden.

Klicken Sie dann auf OK. Der Resturlaub ist nun bei jeder Person korrigiert.



Feldbeschreibungen

Jahr:

In diesem Feld wird die Jahreszahl angegeben.

Urlaub pro Jahr:

Zeigt den Urlaub des Mitarbeiters pro Jahr (in Tagen).

Übertrag aus Vorjahr:

Urlaubsübertrag aus dem Vorjahr.

Url. gen.:

Anzeigefeld für genommenen Urlaub im aktuellen Jahr.

Aktiv:

Anzeige, ob ein Sonderprogramm aktiv oder inaktiv ist.


Jahr

Durch Klick auf diesen Button werden die Änderungen durchgeführt.
