Erfassung > Int. Mitarbeiter > Wochen- und Überstunden

Überstunden (+/-) eines Mitarbeiters zu einem bestimmten Zeitpunkt setzen


Um etwaige Überstunden eines Mitarbeiters aufzuzeichnen, werden hier der Startzeitpunkt (Datum) und die Start-Stundenzahl der Berechnung festgelegt. Ab diesem Zeitpunkt wird die festgelegte Soll-Arbeitszeit mit den Ist-Zeiten der Mitarbeiter aus der Zeiterfassung verglichen. Das Ergebnis entspricht den Überstunden des Mitarbeiters.




Personenauswahl

Auswahlkriterien festlegen zur Filterung der aufzulistenden Personen, für die die Wochen- und Überstunden angezeigt und/oder festgelegt werden sollen.



Feldbeschreibungen

Suchtext:

In diesem Feld wird die Zeichenkette eingegeben, nach der gesucht werden soll.

Zeitspanne von:

In diesem Feld wird angezeigt, von welchem Zeitpunkt an Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Aktive Mitarbeiter:

Mit dieser Checkbox kann ausgewählt werden, ob alle Mitarbeiter angezeigt werden sollen, die den Auswahlkriterien entsprechen, oder nur die Mitarbeiter, die einen anzeigbaren aktiven Datensatz in diesem Zeitraum vorweisen.

Checkbox angehakt = Nur Mitarbeiter, die einen anzeigbaren aktiven Datensatz haben.

Abteilung:

Gibt die Abteilung an, in welcher der Mitarbeiter arbeitet.

Zeitspanne bis:

In diesem Feld wird angezeigt, bis zu welchem Zeitpunkt die Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Historie anzeigen:

Ist diese Checkbox angehakt wird neben den normalen Daten auch ein zusätzlicher Historienverlauf mit angezeigt.


Anzeigen

Durch Klick auf diesen Button wird Liste erzeugt und angezeigt. Je nach festgelegten Auswahlkriterien wird die Liste hierbei entsprechend gefiltert.

Feldbeschreibungen

Mitarbeiter:

Zeigt an, welche Mitarbeiter einem Projekt als Arbeitskräfte zugeteilt sind.

Bei (Teil-)Projekten stehen hier alle Mitarbeiter, die in den jeweiligen Unteraufgaben verteilt sind.

EDat.:

In dieses Feld muss das Eintrittsdatum einer Person, d.h. seit wann der Mitarbeiter bei der Firma angestellt ist, eingetragen werden.

Std/W:

In diesem Feld wird die Anzahl der zu leistenden Arbeitsstunden eines Mitarbeiters (pro Woche) und an welchen Tagen der Mitarbeiter als Ressource verfügbar ist angezeigt.

Monat:

In diesem Feld wird der Kalendermonat über ein Auswahlmenü ausgegeben.

Jahr:

In diesem Feld wird die Jahreszahl angegeben.
