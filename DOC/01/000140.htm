Auflistung einzelner Datenbanktabellen in fx-project®


Der Aufbau einzelner Datenbanktabellen in fx-project kann hier auf Feldebene mit Attributen und Beschreibung aufgelistet werden.




Tabelle auswählen

Tabelle auswählen
Wählen Sie eine Tabelle aus.



Feldbeschreibungen

Tabelle:

In Maske dieses Feld ist ein Auswahlfeld um eine bestimmte Individuell -Tabelle zu selektieren.

Wenn keine individuelle Tabelle ausgewählt ist, wird Textdatei für alle individuelle Tabelle erzeugt. Alias zu ID_Tabellen.

Tabellenart:

Art der Tabelle.
1 - SystemTabelle,
2 - Daten,
3 - Test,
4 - Individuell.

Komplett:

Durch einen Klick auf diesen Button wird der komplette Inhalt aufgelistet.


Anzeigen




Definition der Tabellenfelder

Definition der Felder einer Tabelle.

Es werden alle Felder einer Tabelle angezeigt.
Ebenso ist ersichtlich, in welchen Tabellen das jeweilige Feld noch benutzt wird.
Gibt es zu einem Feld Alias-Namen, so ist dieser in der jeweiligen Tabelle in Klammern hinterlegt. Der Tabellenname ist mit einem Link hinterlegt und kann über ein neues Fenster angezeigt werden.



Feldbeschreibungen

Pri.:

Primärschlüssel in entsprechender Tabelle.

Pos.:

Position dieses Wertes in der Auflistung, d.h. an welcher Stelle der Eintrag stehen soll.

Feldname:

Gibt den Name des Feldes an.

Bei Anlage eines Feldes muss sichergestellt sein, dass der Name eindeutig ist.

Feldtyp:

Im Falle von "DATUM" und "ZEIT" bestimmt dieses Feld, wie der Maskengenerator das entsprechende Feld aufbereitet. Dabei muss für das aufzubereitende Feld mittels SQL ein Aliasfeld definiert werden.

Länge:

Länge des Feldes. Bei den meisten Felder gelten hier gewisse Beschränkungen (z.B. Text maximal 256 Zeichen lang).

Wird eine dieser Beschränkungen verletzt, übernimmt das Programm die Korrektur automatisch.

NNull:

Dieses Feld muss unbedingt ausgegeben werden,

Index:

leer

Grp.:

Zeigt an, ob das Feld doppelte Werte enthalten darf.

Beschreibung:

Erläuternde Beschreibungen, Detailangaben, Bemerkungen oder ähnliches.
