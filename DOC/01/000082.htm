Suchen und Auflisten von bestimmten Personen


Personen werden nach frei definierten Mustern gesucht und gefunden. Diese Muster können sich zusammensetzen aus Name, Personentyp, Stellenbezeichnung oder Abteilung.




Personen suchen

Personen suchen

Hier können die Suchkriterien für die Personensuche eingegeben werden. Je detaillierter die Angaben sind, desto genauer ist die Treffermenge.

HINWEIS:
Bleiben alle Felder leer, dann erhalten Sie alle im System eingebundenen Personen, was allerdings, abhängig von der jeweiligen Datenmenge, zu längeren Wartezeiten führen könnte.



Feldbeschreibungen

Name:

In diesem Feld wird der Familienname der Person oder die Bezeichnung der juristischen Person (z.B. Firmenname) angezeigt bzw. eingetragen.

Personen-Typ:

Kategorisierung der Personen, zu unterscheiden sind z.B. Mandanten, Mitarbeiter, Kunden etc.

Vorname:

In diesem Feld wird der Vorname einer Person angezeigt bzw. eingegeben, sofern das Feld leer ist (maximal 40 Zeichen sind erlaubt).

Pos. Stellenb.:

In diesem Feld wird die Position einer Person in einem Unternehmen laut Stellenbeschreibung ausgewählt, z.B. "Geschäftsleitung", "Verwaltung", "Projektleiter" etc. Dieses Feld beinhaltet also lediglich die Stellenposition, unter der eine Person eingestellt worden ist.

Jeder Person können aber außerdem unterschiedliche Mitarbeiterrollen in der entsprechenden Fähigkeiten- Maske zugeordnet werden, so kann z.B. ein "Geschäftsleiter" auch oder insbesondere die Fähigkeit "Projektleiter" erhalten und dann dem entsprechend in einem Projekt als solcher eingesetzt werden.

Damit ein Projekt angelegt werden kann, muss zumindest einer Person die Mitarbeiterrolle "Projektleiter" zugeordnet werden.

Kontaktmöglichkeit:

Durch Aktivieren dieser Checkbox wird die Suche auf die Kommunikationsdaten der gefundenen Personen ausgedehnt. Andernfalls werden nur Personendaten, wie Name oder Position in der Firma, und Adressdaten angezeigt.

Primärschlüssel in der Tabelle "Abteilung".:

Gibt die Abteilung an, in welcher der Mitarbeiter arbeitet.


Suchen

Durch Klick auf diesen Button wird die Suche gestartet.




Treffer der Personensuche

Treffer der Personensuche

Diese Aufzählung enthält alle Personen, die den Suchkriterien entsprechen. Außerdem werden deren wichtigste Daten angezeigt.
Durch Anhaken der Checkbox Kontaktmöglichkeit erhalten Sie zusätzlich Kommunikationsdaten zu den gefundenen Personen.



Feldbeschreibungen

Name:

In diesem Feld wird der Familienname der Person oder die Bezeichnung der juristischen Person (z.B. Firmenname) angezeigt bzw. eingetragen.

Vorname:

In diesem Feld wird der Vorname einer Person angezeigt bzw. eingegeben, sofern das Feld leer ist (maximal 40 Zeichen sind erlaubt).

Pers.-Typ:

Kategorisierung der Personen, zu unterscheiden sind z.B. Mandanten, Mitarbeiter, Kunden etc.

Abteilung:

Gibt die Abteilung an, in welcher der Mitarbeiter arbeitet.

Stellenp.:

In diesem Feld wird die Position einer Person in einem Unternehmen laut Stellenbeschreibung ausgewählt, z.B. "Geschäftsleitung", "Verwaltung", "Projektleiter" etc. Dieses Feld beinhaltet also lediglich die Stellenposition, unter der eine Person eingestellt worden ist.

Jeder Person können aber außerdem unterschiedliche Mitarbeiterrollen in der entsprechenden Fähigkeiten- Maske zugeordnet werden, so kann z.B. ein "Geschäftsleiter" auch oder insbesondere die Fähigkeit "Projektleiter" erhalten und dann dem entsprechend in einem Projekt als solcher eingesetzt werden.

Damit ein Projekt angelegt werden kann, muss zumindest einer Person die Mitarbeiterrolle "Projektleiter" zugeordnet werden.

Straße:

In dieses Feld wird die Straße mit Hausnummer eingetragen.

PLZ:

In dieses Feld wird die Postleitzahl des Ortes eingetragen.

Ort:

In dieses Feld wird der Wohnort der Person eingetragen.

Land:

In diesem Feld wird ein Länderkennzeichen aus einer Ländertabelle ausgewählt.

Das Länderkennzeichen entspricht dabei den aktuell gültigen Autokennzeichen, z.B. "USA", "F", "NL", "D" etc.

Kommunikationstyp:

In diesem Feld wird aus einem Auswahlmenü die Art der Kommunikationsverbindung, wie z.B. Telefon, Fax, E-Mail etc. ausgewählt.

Nummer:

In diesem Feld wird die eigentliche Kommunikationsverbindung, wie z.B. Telefonnummer, Faxnummer, E-Mail-Adresse etc. eingetragen bzw. angezeigt. Die Eingaben in diesem Feld sind abhängig vom vorher ausgewählten Kommunikationstyp.
