Administration > Profile/Benutzer > Profilprogrammzuordnung

Zuordnen einzelner Programmfunktionen zu einem Profil mit Definition der Rechte (Anzeigen, Neuanlage, Ändern und Löschen)


Hier können Programmfunktionen (einzelne Programme) zu Benutzerprofilen zugeordnet werden. Wird z.B. ein Mitarbeiterprofil angelegt, werden die einzelnen Programmfunktionen per Mausklick zugeordnet. Dabei ist es möglich, nicht nur die Programmfunktion zuzuordnen, sondern auch bei jeder Programmfunktion die einzelnen Verarbeitungsmodi zuzuordnen (soweit vorhanden), z.B. Anzeigen, Neuanlage, Ändern oder Löschen.




Profile wählen

Auswählen einer oder mehrerer Profile, zu denen Programmfunktionen zugewiesen werden sollen.



Feldbeschreibungen

Benutzerprofilgruppe:

In diesem Feld kann man über ein Auswahlmenü die gewünschte Benutzerprofilgruppe selektieren.


Auswählen

Durch einen Klick auf diesen Button werden die ausgewählten Daten weitergeleitet.




Programmberechtigungen festlegen

Hier werden alle Programmfunktionen angezeigt.
Zu den ausgewählten Profilen können Berechtigungen für einzelne Programmfunktionen zugewiesen werden.
Programmfunktionen, die mit blauem Text angezeigt werden, sind Programmfunktionen, die von einem User des Systems mit Hilfe des Programms TR 201 erstellt wurden.



Feldbeschreibungen

Programmname:

Name der Programmfunktion
