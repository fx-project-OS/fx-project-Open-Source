Administration > Profile/Benutzer > Benutzerprofilzuordnung

Zuordnen definierter Profile mit Programmfunktionen zu einem Benutzer oder umgekehrt


Einem Benutzer können beliebig viele Benutzerprofile zugeordnet werden. Alternativ besteht die Möglichkeit, einem Benutzerprofil die entsprechenden Benutzer zuzuordnen. Um ein Profil zuzuordnen, müssen die notwendigen Lizenzpunkte des Profils auf dem Punktekonto aktiviert werden. Wird einem Benutzer ein Profil entzogen, so wird die zugeordnete Anzahl Lizenzpunkte des Profils dem Punktekonto wieder gutgeschrieben. Sind gleiche Programmfunktionen in verschiedenen Profilen (Menge von zugeordneten Programmfunktionen) vorhanden, werden die dazugehörigen Lizenzpunkte nur einmal aktiviert, solange das Programm in einem der aktivierten Profile vorhanden ist.




Auswahlmaske

Benutzer bzw. Benutzergruppen auswählen.

In der Auswahlmaske wählt man den gewünschten Benutzer bzw. Benutzergruppe, zu dem in der Zuordnungsmaske Benutzergruppen bzw. Benutzer zugeordnet werden.

Benutzer können nur zu aktiven Gruppen zugeordnet werden.

HINWEIS:
Die Einträge für das Feld "Benutzerprofilgruppe" können unter dem Menüpunkt "Administration/Benutzerverwaltung/Benutzer definieren" verwaltet werden.
Die Einträge für das Feld "Benutzerprofilgruppe" können unter dem Menüpunkt "Administration/Benutzerverwaltung/Benutzerprofilgruppen definieren" verwaltet werden.
Falls Sie auf eines dieser Programme kein Recht haben bzw. es nicht in Ihrem Hauptmenü erscheint, dann wenden Sie sich bitte an Ihren Administrator, Projektleiter oder Vorgesetzen.

In der Zuordnungsmaske werden dem in der Auswahlmaske gewählten Benutzer bzw. Benutzergruppe die gewünschten Benutzergruppen bzw. Benutzer zugeordnet.

Benutzer können nur zu aktiven Gruppen zugeordnet werden.
