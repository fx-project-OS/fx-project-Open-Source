Graphische Aufbereitung der Zeitdaten nach Fähigkeiten gruppiert


Statistische Auswertung der Fähigkeitenverteilung auf jede einzelne Aufgabe oder Projekt. Hierbei können die Auswahlkriterien komfortabel angepasst werden, um entweder eine globale Projektauflistung zu erhalten oder eine individuelle Auflistung für einen bestimmten Mitarbeiter in einem bestimmten Zeitraum.




Anzeigekriterien für die Projektstatistiken

In der Maske Anzeigekriterien für die Projektstatistiken wird die Auswahl getroffen, welche statistische Auswertung der Fähigkeitsverteilung aufgelistet werden soll.

Es muss mindestens in einem der Felder Projekt, Person, Projektleiter oder Fähigkeit ein spezifizierter Eintrag ausgewählt werden, welcher als Basis zur Ermittlung der Statistikdaten dient. Eine Auswahl von mehreren Einträgen schränkt die ermittelte Datenmenge dementsprechend ein.

Der anzuzeigende Zeitraum kann über die beiden Eingabefelder Zeitspanne von und Zeitspanne bis angegeben werden. Wird keine Eingabe getätigt wird der gesamte relevante Zeitraum einkalkuliert.

Durch Anhaken der Checkbox Komplett werden die ermittelten Fähigkeiten für jede relevante Aufgabe detailliert angezeigt.

Die gewünschten Daten werden nach einem Klick auf den Button Anzeigen ermittelt und in der Maske Projektstatistiken aufbereitet angezeigt. Dort können über den [Datei-Manager] weiterverarbeitet werden.



Feldbeschreibungen

Projekt:

Im Feld Projekt werden die Vorgänge (Projekte oder Aufgaben) aufgelistet, für die man die entsprechenden Rechte besitzt und die entsprechen Filterkriterien gelten.

Person:

In diesem Auswahlfeld wird die Person gewählt, deren Daten je nach Aktion bearbeitet oder angezeigt werden soll.

Zeitspanne von:

In diesem Feld wird angezeigt, von welchem Zeitpunkt an Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Projektleiter:

In diesem Feld wird der Leiter eines Projektes angezeigt bzw. muss aus einer Auswahlliste gewählt werden. Der Projektleiter wird einmal im Mutterprojekt festgelegt, und wird dann als VORSCHLAG in alle darunter liegenden Projekten vorausgewählt.

Grün angezeigte Personen im Auswahlfeld sind Projektleiter, welche bereits mindestens einem Projekt in dieser Rolle zugeordnet sind.

Angelegt werden Projektleiter über die Rollen-Maske mit Rolle "Projektleiter".

Zeitspanne bis:

In diesem Feld wird angezeigt, bis zu welchem Zeitpunkt die Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Fähigkeit:

Auswahl einer bestimmten Fähigkeit bzw. einer Fähigkeitsgruppe zur Gruppierung der Mitarbeiter.

Komplett:

Durch Anhaken dieser Checkbox werden die ermittelten Fähigkeiten detailliert angezeigt für jede relevante Aufgabe.


Anzeigen




Projektstatistiken

Anzeige detaillierter Informationen der Fähigkeitenverteilung zu einem ausgewählten Projekt/Mitarbeiter.

In der Maske Projektstatistiken werden alle wichtigen Informationen bzgl. der Fähigkeitenverteilung zu einem ausgewählten Projekt oder Mitarbeiter übersichtlich dargestellt. Dabei kann der Informationsgehalt variieren je nach eingestellten Anzeigeoptionen (siehe Hilfe zur Maske Anzeigekriterien für die Projektstatistiken)
