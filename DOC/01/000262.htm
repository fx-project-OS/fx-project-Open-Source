Reporting > Personen > Personal-Projekt-Einsatzplan

Graphische Anzeige aller zugeordneten Aufgaben eines Mitarbeiters mit geplanten Soll-Aufwänden


Dieses Diagramm zeigt den Einsatz eines Mitarbeiters (unter Berücksichtigung der vertraglichen Arbeitszeit) in seinen verschiedenen Projekten.
Hierbei wird veranschaulicht, dass ein Mitarbeiter in mehreren Projekten zur gleichen Zeit eingebunden sein kann.




Kriterienauswahl

Kriterienauswahl für die Darstellung im Personal-Projekt-Einsatzplan

In der Auswahlmaske wird der Mitarbeiter ausgewählt, zu dem die Auslastung im entsprechenden Zeitraum (Zeitspanne von - Zeitspanne bis) angezeigt werden soll.

Als Anzeigeart kann "automatisch", "Tage", "Wochen", "Monate" oder "Jahre" eingestellt werden. Bei Auswahl von "automatisch" wird die für die Diagrammgröße geeignetste Anzeigeart eingestellt. Dieser Auswahlwert ist verantwortlich für die Einteilung der X-Achse.

Wird "Std. aufaddieren" angehakt, werden alle Stunden des Mitarbeiters pro Zeiteinheit übereinander aufaddiert angezeigt, ansonsten wird pro Aufgabe ein Eintrag nebeneinander angezeigt.

Wird "keine" Ebene eingegeben, werden alle Aufgaben dieser Person in diesem Zeitraum angezeigt, ansonsten werden alle untergeordneten Aufgaben bis zur angegebenen Ebene aufaddiert.



Feldbeschreibungen

Zeitspanne von:

In diesem Feld wird angezeigt, von welchem Zeitpunkt an Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Person:

In diesem Auswahlfeld wird die Person gewählt, deren Daten je nach Aktion bearbeitet oder angezeigt werden soll.

Projekt:

Im Feld Projekt werden die Vorgänge (Projekte oder Aufgaben) aufgelistet, für die man die entsprechenden Rechte besitzt und die entsprechen Filterkriterien gelten.

Zeitspanne bis:

In diesem Feld wird angezeigt, bis zu welchem Zeitpunkt die Daten für ein Projekt aufgelistet werden sollen.

Im Eingabemodus kann hier ein Datum entsprechend den Benutzereinstellungen eingegeben werden.

Kal.-Anzeige:

Dient zur Festlegung der Kalenderzeitabstände pro Spalte, d.h. Tage, Wochen, Monate oder Jahre.

Falls "Automatisch" festgelegt wurde wird der Kalenderzeitabstand programmtechnisch ermittelt anhand der Länge des gesamt anzuzeigenden Zeitraums.

Ebene:

Die Zahl die Sie in diesem Feld angeben steuert, wie viele Projektebenen im Report angezeigt werden sollen.

Unterebene "0" entspricht der höchsten Ebene.

Zeit aus Projekt:

Ist diese Checkbox angehakt, dann wird die eingegebene Zeitspanne automatisch überschrieben mit den Zeitdaten aus dem ausgewählten Projekt.

Std. aufadd.:

Wird diese Checkbox angehakt, dann werden alle Stunden des Mitarbeiters pro Zeiteinheit übereinander aufaddiert angezeigt, ansonsten wird pro Projekt ein Eintrag schräg versetzt hintereinander angezeigt.

Unbekannt anz.:

Falls diese Checkbox angehakt ist werden auch "unbekannte" Aufgaben angezeigt.


Anzeigen

Durch Klick auf diesen Button wird Liste erzeugt und angezeigt. Je nach festgelegten Auswahlkriterien wird die Liste hierbei entsprechend gefiltert.




Diagramm

Diagramm Erläuterungen zum Personal-Projekt-Einsatzplan
Vor dem heutigen Datum wird im Diagramm eine vertikale rote Linie gezogen. Alles was links von der Linie steht entspricht den tatsächlichen Ist-Daten, alles was rechts von der Linie steht den geplanten Soll-Daten.

In einer grünen horizontalen Linie werden die Sollstunden des Mitarbeiters pro Zeiteinheit der x-Achse (Stunden, Wochen, Monate, Jahre) angezeigt.

Auf der y-Achse (Hochwert) wird die Arbeitszeit (= Potential) in Stunden (h) angezeigt, auf der x-Achse (Rechtswert) die Kalenderzeit.

Für die Darstellung werden ca. 10 Farben vorgegeben. Sollten mehr als 10 Farben benötigt werden dann werden die Farben wiederholt mit einer leichten Helligkeitsänderung.

HINWEIS:
Achtung: Bei vielen Projekten kann die Farbe sehr schnell nicht mehr unterscheidbar werden. Bitte schränken Sie in so einem Fall die angezeigten Daten mittels Filter, Zeitraum etc. ein!
