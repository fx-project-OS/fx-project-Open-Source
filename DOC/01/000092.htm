Graphische Übersicht (in Abhängigkeit von Zugriffsrechten) sämtlicher Mitarbeiter über deren Urlaubs-, Schulungs-, Krankheitstage und/oder Ist-Zeiten


Graphische Übersicht sämtlicher Mitarbeiter über deren Urlaubs-/Schulungs- bzw. Krankheitstage.




Grafische Übersicht über Abwesenheitstage und/oder gearbeitete Ist-Stunden

Über diesen graphischen Report haben Sie Einblick, welche Mitarbeiter wann Urlaub geplant oder genommen haben, wer gerade auf Schulung unterwegs ist, oder wer sich krank gemeldet hat. Sie sehen, auf wie viele Ressourcen Sie in einem gewissen Zeitraum verzichten müssen.

In den beiden Feldern Zeitspanne von und Zeitspanne bis müssen Sie jeweils ein Datum angeben, d.h. für welchen Zeitraum der graphische Report erstellt werden soll.

Im Feld Kalenderanzeigemodus können Sie auswählen, wie detailliert der Report aufbereitet werden soll, d.h. ob Sie Tage, Wochen, Monate oder Jahre sehen wollen. Im Modus "Automatisch" ermittelt das Programm die Darstellungsweise in der Art, dass sämtliche Informationen noch in das Browserfenster passen.

Durch Anhaken der Felder Urlaub, Krank oder Schulung kann bestimmt werden, welche Arten von Fehltagen angezeigt werden sollen. WICHTIG! Es muss mindestens ein Eintrag angehakt werden.

Im Auswahlfeld Abteilung können Sie bestimmen, ob Sie die graphische Übersicht für alle Mitarbeiter sehen wollen, oder nur für eine spezielle Abteilung.

Durch Anhaken des Feldes Aktive Mitarbeiter werden in der Auflistung nur die Mitarbeiter angezeigt, die im ausgewählten Zeitraum Fehltage haben. Ist dieses Feld nicht angehakt, dann werden alle Mitarbeiter angezeigt.

HINWEIS:
Falls Sie auf eines dieser Programme kein Recht haben bzw. es nicht in Ihrem Hauptmenü erscheint, dann wenden Sie sich bitte an Ihren Administrator, Projektleiter oder Vorgesetzen.



Feldbeschreibungen

Von:

In dieses Feld muss ein Datum eingegeben werden, ab welchem Zeitpunkt die grafische Darstellung des USK-Plans beginnen soll.
(Das Datum muss in dem Format eingegeben werden, welches in den Benutzereinstellungen definiert wurde.)

Kal.-Anzeige:

Dient zur Festlegung der Kalenderzeitabstände pro Spalte, d.h. Tage, Wochen, Monate oder Jahre.

Falls "Automatisch" festgelegt wurde wird der Kalenderzeitabstand programmtechnisch ermittelt anhand der Länge des gesamt anzuzeigenden Zeitraums.

Urlaub:

Durch Anhaken dieser Checkbox werden die Urlaubstage der ausgewählten Mitarbeiter grafisch aufbereitet dargestellt.

Schulung:

Durch Anhaken dieser Checkbox werden die Schulungstage der ausgewählten Mitarbeiter grafisch aufbereitet dargestellt.

Krank:

Durch Anhaken dieser Checkbox werden die Krankheitstage der ausgewählten Mitarbeiter grafisch aufbereitet dargestellt.

Gleittag:

Dieses Feld gibt an, ob der Arbeiter berechtigt ist in Gleittag zu arbeiten.

Ist der User dazu berechtigt, dann kann er innerhalb der Zeiterfassung Zeitdatensätze auf "Gleittag" buchen.

Dies ermöglicht dem User, z.B. für Freitag Gleittag einzutragen, wenn er bereits bis Donnerstag sein Soll erfüllt hat.

Indem der User diesen Eintrag macht wird zusätzlich verhindert, dass er über das Zeiterfassungs-Check-Programm registriert wird, obwohl er bereits die nötigen Stunden erbracht hat.

IST-Std.:

Gibt an, wie viele Stunden ein Mitarbeiter in einem bestimmten Zeitraum tatsächlich gearbeitet hat.

Bis:

In dieses Feld muss ein Datum eingegeben werden, bis zu welchem Zeitpunkt die grafische Darstellung des USK-Plans erfolgen soll.
(Das Datum muss in dem Format eingegeben werden, welches in den Benutzereinstellungen definiert wurde.)

Abteilung:

Hier kann die Abteilung und dementsprechend die der Abteilung zugehörigen Mitarbeiter ausgewählt werden, zu welcher man die Informationen benötigt. Wird keine Abteilung ausgewählt, dann erhält man die Informationen zu allen Mitarbeitern.

Aktive MA:

Mit dieser Checkbox kann ausgewählt werden, ob alle Mitarbeiter angezeigt werden sollen, die den Auswahlkriterien entsprechen, oder nur die Mitarbeiter, die einen anzeigbaren aktiven Datensatz in diesem Zeitraum vorweisen.

Checkbox angehakt = Nur Mitarbeiter, die einen anzeigbaren aktiven Datensatz haben.


Anzeigen

Durch Klick auf diesen Button wird Liste erzeugt und angezeigt. Je nach festgelegten Auswahlkriterien wird die Liste hierbei entsprechend gefiltert.
