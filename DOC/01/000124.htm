Export von Projekten aus fx-project® als MS-Project® CSV-Datei


Erzeugt aus dem gewählten Projekt bzw. Teilprojekt eine CSV-Datei, welche in MS-Project® eingelesen werden kann. Dadurch können Daten sehr einfach zwischen fx-project und MS-Project® ausgetauscht werden.


HINWEIS:
Die verwendeten Felder der CSV-Datei können frei definiert und angepasst werden. Die Standardwerte sind jedoch so ausgelegt, dass ein manuelles Anpassen an MS-Project® nicht notwendig ist.




Projekte nach MS-Project® exportieren

Projekte nach MS-Project® exportieren

Hier lassen sich Projekte aus fx-project nach MS-Project® exportieren (als Textdatei mit Tabulator getrennt).

  • Wählen Sie ein Projekt aus und geben Sie an, in welchem Format das Datum und in welcher Einheit der Aufwand in MS-Project® gewünscht wird. Klicken Sie dann auf den Download-Button, um das File downzuloaden.
    Nach MS-Project® werden folgende Werte übernommen:
    • Nummer
    • Projektbezeichnung
    • Soll-Beginn
    • Soll-Ende
    • Art des Projektes
    • Nachfolger
    • Aufwand Soll
    • Beschreibung
    • Priorität
    • Fertigstellungsgrad
    • Position
    • Ebene
    • Erstelldatum


    Nach dem Export öffnen Sie die Datei mit MS-Project® und wählen eine passende MS-Project® Import-Abbildung (oder legen Sie eine neue Datei an, siehe MS-Project® Hilfe).



    Feldbeschreibungen

    GIF-Bild:

    Gibt an welches Icon bei diesem Eintrag verwendet wird.

    Auswahl des Vorgangs (Projekt, Teilprojekt, Aufgabe) für den man berechtigt ist.:

    Im Feld Projekt werden die Vorgänge (Projekte oder Aufgaben) aufgelistet, für die man die entsprechenden Rechte besitzt und die entsprechen Filterkriterien gelten.

    Ziel (Datei):

    Ziel (Datei, Projekt etc.) auswählen

    Teilprojekte:

    Bei aktivierter Checkbox werden neben dem Hauptprojekt auch sämtliche untergeordneten Teilprojekte und Aufgaben einbezogen.

    ACHTUNG!
    Dies kann bei großen Projekten zu längeren Wartezeiten führen.

    Trennzeichen:

    Trennzeichen für die Exportfelder bei Projektexport in MS_Projekt.

    Datumsformat:

    Legt das Format fest, in dem alle Datumsangaben angezeigt und eingelesen werden. Das Datum muss mit folgenden Variablen definiert werden. Trennzeichen beliebig:

    dd: Tag (day) 2-stellig
    mm: Monat 2-stellig
    yy: Jahr (Year) 2-stellig
    yyyy: Jahr 4-stellig

    Prefix entfernen:

    Prefix (dt. Präfix oder auch Vorsilbe genannt) bezeichnet ein dem Wortstamm vorangestellte Zeichenfolge, hier um eine Gruppe von zusammengehörigen Wörtern zu erzeugen.

    Wird z.B. beim Projektnamen verwendet um eine einheitliche Projektzugehörigkeit bereits beim Namen zu erreichen. Der hierbei zu verwendende Platzhalteraufbau kann in den Administrator-Einstellungen definiert werden.

    Ressourcen kopieren:

    Ist dieses Feld angehakt, dann werden auch die Ressourcen, die einer Aufgabe zugeteilt sind, mitkopiert.

    Feldbeschreibungen

    Status:

    In diesem Feld wird angezeigt, wie weit ein Projekt bereits fortgeschritten ist. Im Eingabemodus kann man hier einem Projekt ein Attribut aus einer Auswahlliste zuordnen, das wiedergibt, ob bzw. wie weit ein Projekt bereits fortgeschritten ist.

    Auch kann über den Status entschieden werden, ob Projektdaten in einen Report einfließen sollen oder nicht.

    Projekt:

    In diesem Feld wird der Projektname des aktuell ausgewählten Projektes angezeigt.

    Zeitraum:

    Zeigt in einem kombinierten Feld die Laufzeit des Projekts an.

    Aufw. Soll:

    Der Soll-Aufwand entspricht dem geplanten Zeitaufwand in Stunden, Manntagen oder Mannjahren etc. Der geschätzte Aufwand wird bei der Definition von Aufgaben eingetragen. Bei Projekten/Teilprojekten entspricht dieser Wert der Summe aller untergeordneten Aufgaben.

    Prio.:

    Gibt die Dringlichkeit des (Teil-)Projektes an.
    Je höher die Priorität eines Projektes ist, desto höher wird dieser im Kontrollzentrum eines Mitarbeiters angezeigt.
